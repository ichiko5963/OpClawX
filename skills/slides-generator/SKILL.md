# Slides Generator

CanvaやGoogle Slidesでプレゼンテーションを自動作成する。

## 必要な情報

- スライドのタイトル
- スライドの内容（テキスト、ポイント）
- 使用するテンプレート（ある場合）
- 保存先（Google Driveのフォルダ）

## 作業フロー

### 1. Canvaを使用する場合
- Canva API 또는 网站で新規プレゼンテーションを作成
- テンプレートを選択或は白紙から作成
- スライド数を指定
- テキストや画像を挿入

### 2. Google Slidesを使用する場合
- Google Slides API或はGoogle Driveに保存
- テンプレートからコピー
- スライドを編集

### 3. 保存・共有
- Google Driveに適切なフォルダに保存
- 共有リンクを取得（全公開或は限定共有）

### 4. ユーザーに結果を報告
- スlidesのリンクを報告
