# 請求書テンプレート（画像準拠・複数行明細対応）

## 画像テンプレートの項目

```
================================================================================
                              請求書
================================================================================

請求番号: {{invoice_no}}
請求日: {{date}}

宛先: {{company_name}}
件名: {{title}}
================================================================================

--------------------------------------------------------------------------------
品名                    数量    単価        金額
--------------------------------------------------------------------------------
{{item_1_name}}         {{item_1_quantity}}    {{item_1_unit_price}}   {{item_1_amount}}
{{item_2_name}}         {{item_2_quantity}}    {{item_2_unit_price}}   {{item_2_amount}}
{{item_3_name}}         {{item_3_quantity}}    {{item_3_unit_price}}   {{item_3_amount}}
...（最大10行）

                                          小計:    {{subtotal}}
                                          消費税:  {{tax}}
                                          合計:    {{total}}
                                          
================================================================================
支払期限: {{due_date}}
================================================================================

【備考】
{{details}}

================================================================================
                      お取引ありがとうございます
================================================================================
```

## テンプレート変数一覧

### 基本情報
| 変数 | 例 |
|------|-----|
| `{{invoice_no}}` | 026, INV-2026-0024 |
| `{{date}}` | 2026年1月31日 |
| `{{company_name}}` | Tailent株式会社 |
| `{{title}}` | AIライティングツール制作 |

### 明細行（最大10行）
| 行 | 品名 | 数量 | 単価 | 金額 |
|----|------|------|------|------|
| 1 | `{{item_1_name}}` | `{{item_1_quantity}}` | `{{item_1_unit_price}}` | `{{item_1_amount}}` |
| 2 | `{{item_2_name}}` | `{{item_2_quantity}}` | `{{item_2_unit_price}}` | `{{item_2_amount}}` |
| ... | ... | ... | ... | ... |
| 10 | `{{item_10_name}}` | `{{item_10_quantity}}` | `{{item_10_unit_price}}` | `{{item_10_amount}}` |

### 計算結果
| 変数 | 例 |
|------|-----|
| `{{subtotal}}` | ¥100,000 |
| `{{tax}}` | ¥10,000 |
| `{{total}}` | ¥110,000 |

### 支払情報
| 変数 | 例 |
|------|-----|
| `{{due_date}}` | 2026年2月28日 |
| `{{details}}` | 備考欄 |

## 使用例（画像と同じ構成）

**入力:**
```python
items = [
    {'name': '記事制作費', 'quantity': 1, 'unit_price': 80000},
    {'name': 'オプション', 'quantity': 1, 'unit_price': 20000},
]
company_name = 'Tailent株式会社'
title = 'AIライティングツール制作'
due_date = '2026年2月28日'
invoice_date = '2026年1月31日'
```

**結果:**
- 小計: ¥100,000
- 消費税: ¥10,000
- 合計: ¥110,000
