# YouTube Live Scheduler

YouTubeでライブをスケジュール設定し、限定公開で公開する。

## 必要な情報

- Googleアカウント（AirCleのYouTubeチャンネル）
- ライブの日時
- ライブのタイトル
- 公開設定（限定公開/公開）

## 作業フロー

### 1. YouTubeスタジオにアクセス
- ブラウザで YouTube Studio (studio.youtube.com) を開く
- AirCleのGoogleアカウントでログイン

### 2. ライブ作成画面へ
- 左メニューから「コンテンツ」→「ライブを作成」をクリック
- または「新規」→「ライブストリーム」を選択

### 3. ライブの詳細設定
- **タイトル**: ユーザーが指定したタイトルを入力
- **説明**: 必要に応じて説明を追加
- **公開設定**: 「限定公開」（リンク知道的のみ）或は「公開」を選択
- **スケジュール**: 日時を設定
  - 「スケジュールを設定」をクリック
  - 希望の日時を選択
  - 時間を選択

### 4. 保存・確認
- 「保存」或は「スケジュール」をクリック
- ライブがスケジュールされたことを確認
- **ライブリンクを取得** - 共有用URLをコピー

### 5. ユーザーにリンクを報告
- 取得したライブリンクをユーザーに送信

## エラー時の対応

- ログインが必要な場合: ユーザーにログイン情報を依頼
- ライブ作成権限がない場合: チャンネル権限を確認
- エラーが発生した場合: スクリーンショットを撮り、ユーザーに報告
