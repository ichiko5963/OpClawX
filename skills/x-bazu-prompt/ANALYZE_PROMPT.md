# Xバズ投稿分析シート用プロンプト（短縮版）

## 入力
X Premium CSVまたは分析シート

## 処理
```
1. データ受信
↓
2. 以下を抽出：
   - 高いいね数の投稿トップ5
   - 各投稿のフック、文字数、構造、特徴
   - 共通パターン
↓
3. 分析結果を出力
```

## 出力形式

```
【バズ分析結果】

■ 効果的なフックランキング
1位: [フック名] - 平均[数値]いいね
2位: [フック名] - 平均[数値]いいね
3位: [フック名] - 平均[数値]いいね

■ 最適文字数
平均: [数値]文字
範囲: [数値]-[数値]文字

■ 必須要素
- [要素1]
- [要素2]
- [要素3]

■ 効果的な表現
- [表現1]
- [表現2]

■ トーン傾向
[一言でまとめたトーン]
```

## 注意
- データが少ない場合は「データ不足」と報告
- 明確な傾向がない場合は「傾向不明」と報告
