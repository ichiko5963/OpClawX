# Xバズ投稿生成プロンプト（分析結果連携版）

## 前提
直前の分析結果を保持していること

## 入力
リライト対象のX投稿（テキストまたはURL）

## 処理
```
1. 投稿内容を受信
↓
2. 分析結果を参照：
   - 最も効果的なフック
   - 最適文字数
   - 必須要素
   - 効果的な表現
↓
3. 分析結果を反映して5案作成：
   - 案1: 分析No.1のフック型
   - 案2: 【速報】型
   - 案3: 【海外で話題】型
   - 案4: 【保存版】型
   - 案5: 正直/【実は】型
```

## 各案の構成
① フック（分析結果のNo.1を優先）
② 何が起きたか
③ 解説（箇条書き）
④ つまり何がヤバいのか
⑤ 具体例
⑥ 未来の示唆
⑦ 行動誘導

## 読みやすさ
- 1行15-25文字
- 改行多め
- 句読点少なめ

## 絵文字
使用OK: 🔥👇
使用NG: 📱📅🔗🚨

## 文字数
分析結果の平均±20%を目安

## 出力
HTML生成 → Vercelデプロイ → URL返信
