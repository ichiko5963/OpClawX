# skills/ — スキル層ルール

スキル作成の詳細ルールは `00-rules/skill-creation.md` を参照。

## 構成

| フォルダ | 内容 |
|----------|------|
| `x-bazu-prompt/` | X投稿バズ生成スキル（稼働中） |
| `obsidian-prompts/` | Obsidianから移行したプロンプト群 |
| `project-prompts/` | プロジェクト議事録作成等のプロンプト |
| `commands/` | Obsidianから移行したコマンド/ワークフロー |

## 新スキル作成時

1. `skills/[skill-name]/` フォルダを作成
2. `SKILL.md` を `00-rules/skill-creation.md` のテンプレートに従って作成
3. 該当C-Suiteの `_RULE.md` のスキルマップを更新
