①毎日3〜5投稿する（最低ライン）
→ 量が足りてない人が9割
→ 「何を投稿すればいいかわからない」は甘え
→ 毎日ニュースを読んで、自分の意見を書くだけでいい

②返信しまくる
→ 自分の投稿だけじゃなく、他人の投稿に返信
→ 「参考になりました」は意味ない。具体的な意見を書け
→ 返信から自分のプロフィールに来る人は多い

③プロフィールを月1で見直す
→ 「何をしている人か」が3秒でわかるか？
→ ヘッダー画像、自己紹介文、固定投稿 → 全部最新にしろ
→ プロフィールが弱いと、興味を持った人が離脱する

④「伸びた投稿」を分析する
→ なぜ伸びたか？型は？時間帯は？
→ 伸びたパターンを意識的に再現しろ
→ バズは偶然じゃない。再現可能なフォーマットがある

Grokが重視するシグナル👇

①リアクションの多様性
→ いいねだけじゃなく、RT + リプライ + 引用RTが全部ある投稿が強い
→ 「いいね狙い」だけの投稿は弱くなった

②滞在時間
→ ユーザーが投稿の前で何秒止まったか
→ 長文で「読ませる」投稿は滞在時間が長い
→ 一行の「名言」より、ストーリーのある投稿が有利

③リプライの質
→ リプライの中身もGrokは読んでいる
→ 「いい質問だ」「具体的にどうやる？」などの深いリプライが多い投稿は評価UP
→ スパムリプライが多い投稿は評価ダウン

実践的なアクション:
→ 「○○についてどう思う？」で締めてリプライを誘発
→ ストーリー形式で書いて滞在時間を伸ばす
→ フォロワーとの対話を意識して投稿設計
