## 06 VS Battle / 比較型

### Example 1: Tool Comparison
```
【頂上決戦】Notion vs Obsidian ⚔️
1年間両方使った結論👇

Notionの強み：
・チーム共有が容易
・データベース機能
・初心者でもすぐ使える

Obsidianの強み：
・ローカル保存（安心）
・無料でほぼ全機能
・カスタマイズ性

結論：チームならNotion、個人ならObsidian
```

## Usage Tips / 活用のコツ

### When to use which pattern? / どの型を使うべきか

| Situation | Recommended Pattern |
|-----------|---------------------|
| New product launch | 01 Breaking News |
| Tutorial content | 02 Save for Later |
| Overseas trend | 03 Global Trend |
| Analysis/Opinion | 04 Conclusion First |
| Honest review | 05 Honest Opinion |
| Comparing options | 06 VS Battle |
| First impressions | 07 First Impression |
| Data report | 08 By The Numbers |
| New insight | 09 Insight |
| Giveaway | 10 Free Resource |
| Pro tips | 11 Pro Tips |
| Warning/Alert | 12 Warning |
| Success story | 13 Storytelling |
| Comprehensive guide | 14 Complete Guide |
| Future prediction | 15 Future Forecast |

### Platform Optimization / プラットフォーム別最適化

**X/Twitter:**
- Keep under 280 characters
- Use only 🔥👇 emojis
- No numbered lists (①②③)

**Instagram:**
- Can be longer (up to 2,200)
- Use more emojis
- Add line breaks for readability

**LinkedIn:**
- Professional tone
- Include data/statistics
- Encourage comments

**note:**
- Long-form OK
- Use images/embeds
- Story-focused
