# 06 VS Battle / 比較型

## 🇺🇸 English

### Definition
Compare two options side-by-side, providing decision-making materials. Resolves the reader's "which is better?" dilemma.

### Psychology
- **Decision Support**: Want help choosing
- **Clarity**: Clear comparison reduces anxiety
- **Validation**: Confirming their choice

### Structure
```
【Battle】+ A vs B ⚔️
↓
Each strength (2-3 points each)
↓
Weakness comparison
↓
Recommendation based on user type
```

### Prompt Template
```
Create a "VS Battle" style post comparing {theme}:
- Use 【頂上決戦】or 【徹底比較】
- Compare 3 strengths each
- Clear recommendation
- Use ⚔️ and 👇 only
```

### Example
> 【頂上決戦】Aqua Voice vs Super Whisper ⚔️ Used both for 6 months. Results: Aqua = Japanese accuracy god, Super = English/code specialist. Conclusion: Japanese-focused → Aqua, Global → Super 👇

## 🇯🇵 日本語

### 定義
2つの選択肢を対比し、判断材料を提供。読者の「どっちがいい？」という迷いを解消。

### 心理メカニズム
- **意思決定支援**: 選ぶのを手伝ってほしい
- **明確化**: 比較で不安が減る
- **正当化**: 自分の選択を確認したい

### 構造
```
【頂上決戦】+ A vs B ⚔️
↓
それぞれの強み（2-3点ずつ）
↓
弱みの比較
↓
ユーザータイプ別の推奨
```

### プロンプトテンプレート
```
{theme}について比較型の投稿を作成:
- 【頂上決戦】または【徹底比較】を使用
- それぞれ3つの強みを対比
- 明確な推奨を提示
- ⚔️と👇のみ使用
```

### 例文
> 【頂上決戦】Aqua Voice vs Super Whisper ⚔️ 半年以上両方使ってきた結論👇 ・Aqua Voice: 日本語精度が神 ・Super Whisper: 英語・コードが得意 結論：日本語メインならAqua、グローバル志向ならSuper

### 💡 コツ
- 「結論」を明確に述べる
- 「どんな人にどっちがおすすめ」を具体的に
- 両方の良さを認めた上で推奨する（偏りすぎない）
