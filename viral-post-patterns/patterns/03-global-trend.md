# 03 Global Trend / 海外バズ型

## 🇺🇸 English

### Definition
Introduce overseas trends as "not yet available in Japan" content. Stimulates desire for first-mover advantage and interest in international reputation.

### Psychology
- **First-Mover Advantage**: Want to know before others
- **Social Proof**: Foreign validation matters
- **Exclusivity**: Early access feeling

### Structure
```
【Trending Overseas】+ Subject + is getting attention 🌎
↓
International reception and use cases
↓
Translation and practical applications
↓
Prediction for Japan release
```

### Prompt Template
```
Create a "Global Trend" style post about {theme}:
- Include 【海外で話題】
- Mention overseas reactions/reviews
- Provide Japanese translation
- Practical usage tips
- Use 👀 and 🌎 only
```

### Example
> 【海外で話題】Gemini 3.1 Pro's frontend features are being called "production-ready" 👀 Real developer use cases 👇 Complex UI implementation → Auto-generated code → Bug reduction. Japan release expected Q2 🌎

---

## 🇯🇵 日本語

### 定義
海外のトレンドを「日本未上陸情報」として紹介。先行者利益の欲求と海外の評判への関心を刺激。

### 心理メカニズム
- **先行者利益**: 誰よりも早く知りたい
- **社会的証明**: 海外での評価が気になる
- **希少性**: まだ知られていない特別感

### 構造
```
【海外で話題】+ 対象 + が注目されてる 🌎
↓
海外での評価・活用事例
↓
日本語訳・具体的な活用方法
↓
日本展開の予測
```

### プロンプトテンプレート
```
{theme}について海外バズ型の投稿を作成:
- 【海外で話題】を含める
- 海外の評価・反応を言及
- 日本語での解説を加える
- 具体的な活用事例を提示
- 👀と🌎のみ使用
```

### 例文
> 【海外で話題】Gemini 3.1 Proのフロントエンド機能が「実用レベル」として注目されてる👀 開発者が実際にやっていること👇 複雑なUI実装 → 自動コード生成 → バグ削減。日本展開はQ2予定🌎

### 💡 コツ
- 「まだ日本では」「先行事例」などの言葉を使う
- 海外の具体的な評価（星評価・コメント）を引用
- 日本での展開予想・活用アドバイスを加える
