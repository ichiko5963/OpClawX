# 03-daily ルール

## 構成

| フォルダ | 内容 | 更新頻度 |
|----------|------|---------|
| `00-daily-note/` | 日記（時系列で何が起きたか要素ひとつも漏らさず記載） | 毎日 |
| `01-calendar/` | 月別の主な予定（Googleカレンダーから遊び系を抽出） | 都度 |
| `02-遊びnote/` | 遊んだ記録（いつ誰と遊んだか正確に） | 遊んだ後 |
| `03-ウケたエピソード/` | 面白かった話（前提・流れがわかるように保存） | 録音/日記から |
| `04-改善点/` | 改善すべき点（人を笑わせることが多い等の特徴も含む） | 随時 |

## 00-daily-note ルール

- ファイル名: `YYYY-MM-DD.md`
- 書けてない場合: 20:00にリマインド
- 日々の録音を送信することでも代替可能
- 時系列で何をやったか・何が起きたかを要素ひとつも漏らさず記載

## 02-遊びnote ルール

- ファイル名: `YYYY-MM-DD-[タイトル].md`
- 「いつ」「誰と」「何をした」「何を思った」を正確に記録

## 03-ウケたエピソード ルール

- 録音の中でウケた部分を個別に保存
- 前提情報・話の流れがわかるように構成
- そのまま人に話せる形で残す
