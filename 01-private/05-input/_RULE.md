# 05-input ルール

## 目的

外部からのインプット（学習コンテンツ）を記録する場所。

## 構成

| フォルダ | ソース | 自動化 |
|----------|--------|--------|
| `kindle/` | Kindleハイライト | 毎時自動取得 |
| `youtube/` | YouTube動画メモ | 手動 or 要約ツール |
| `inside-stories/` | インサイドストーリーズ | 手動 |
| `sources/` | Obsidianから移行した参考ノート | 移行済み |

## カテゴリ抽出ルール

- 各コンテンツの中からカテゴリを自動抽出
- 抽出したカテゴリは `06-学び/` の既存カテゴリと照合
- 該当カテゴリがあれば `06-学び/[カテゴリ]/episodes/` にも紐づけ
- 自分の体験と結びつかないものはここに留める

## ファイル命名

- Kindle: `[書名].md`
- YouTube: `[動画タイトル].md`
- インサイドストーリーズ: `[エピソード名].md`
