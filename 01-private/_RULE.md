# 01-private — 個人知識層ルール

プライベートの知識・内面・成長を管理するフォルダ。
仕事関係は `02-business/` で管理する。ここは個人的な領域のみ。

## 構成

| フォルダ | 内容 |
|----------|------|
| `00-信念/` | 価値観・意思決定の癖・好き嫌い |
| `01-口調/` | 話し方のパターン・フィラーワード・改善記録 |
| `02-people/` | プライベートの人間関係（友人・家族） |
| `03-daily/` | 日記・カレンダー・遊び・エピソード・改善点 |
| `05-input/` | インプット記録（Kindle・YouTube・記事等） |
| `06-学び/` | **最重要** — 経験から抽出した学びのカテゴリ管理 |

## セキュリティ

- MEMORY.md同様、メインセッション（直接チャット）でのみアクセス
- グループチャットや共有コンテキストでは参照しない
