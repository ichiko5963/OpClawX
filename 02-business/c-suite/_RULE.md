# C-Suite 横断ナレッジ ルール

## 目的

各C-Suite役割がプロジェクトを超えて蓄積するナレッジの管理。
事業・プロジェクトの成果物から抽出した**汎用的な学び**をここに集約する。

## 各役割のフォルダ構成

```
[role]/
├── _RULE.md           ← 役割固有ルール + スキルマップ
├── skills/            ← その役割の「やり方」の蓄積
├── knowledge/         ← プロジェクト横断の学び・データ
└── source-knowledge/  ← 参照元情報（外部ソース）
```

## フォールバック検索

情報が見つからない場合の検索順序：
1. 該当役割の `knowledge/`
2. 該当役割の `source-knowledge/`
3. `ceo/source-knowledge/` （最終フォールバック）

## ナレッジ更新ルール

- 成果物を保存したら、汎用的な学びを `knowledge/` に追記
- 新しいスキルを発見したら `skills/` に追記し、`_RULE.md` のスキルマップも更新
- 外部からの情報は `source-knowledge/` に格納
- 重要な決定事項は `ceo/decisions.md` にも記録
