# MTG Inbox ルール

## 目的

MTG文字起こしの一時的な受け入れ口。ここに投入されたファイルは自動的に処理・振り分けされる。

## 処理フロー

文字起こしファイルが投入されたら：

1. **内容を解析** — どの事業/プロジェクトのMTGか特定
2. **既存事業のMTG** → 該当事業の `mtg/YYYY-MM-DD-[議題].md` に移動
3. **既存プロジェクトのMTG** → 該当プロジェクトの `mtg/` に移動
4. **新プロジェクトの話題** → `00-rules/project-lifecycle.md` に従いプロジェクト生成
5. **アクションアイテム抽出** → 該当 `backlog.md` に追加
6. **新メンバー検出** → CHRO の `all-members.md` + 該当 `people.md` に追加
7. **財務情報** → CFO の `knowledge/` に追記
8. **営業情報** → CSO の `knowledge/` に追記

## ファイル命名規則

投入時: 任意（受け取ったままでOK）
振り分け後: `YYYY-MM-DD-[議題].md`

## 処理済みファイル

振り分け完了後、このフォルダからは削除される（移動先に保存済み）。
