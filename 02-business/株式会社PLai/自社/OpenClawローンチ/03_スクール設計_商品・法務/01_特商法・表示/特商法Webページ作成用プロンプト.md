# 特定商取引法に基づく表記 Webページ作成用プロンプト

以下の「依頼文」「掲載する情報」「ページ要件」をそのままコピーし、Webページ（HTMLまたはLP内の1ページ）を作成するAI・開発者に渡してください。

---

## 依頼文（プロンプトとして渡す文）

```
特定商取引法に基づく表記のWebページを1ページで作成してください。

【掲載する情報】
下記「掲載情報」に、販売事業者の会社情報（履歴事項全部証明書に基づく）と、特商法で必要な項目・記載内容をすべて記載しています。この内容を漏れなくページに反映してください。

【ページ要件】
- タイトルは「特定商取引法に基づく表記」とする。
- 項目名と記載内容を、見やすい表形式（2列：項目｜記載内容）で表示すること。スマートフォンでも表が崩れず読めること（レスポンシブ対応）。
- 文字サイズは十分に読めること。印刷したときにも読みやすいこと（印刷スタイルの配慮があれば望ましい）。
- デザインはシンプルで、法的な表記として信頼感のある見た目にすること。過度な装飾は不要。
- フッターに「最終更新日：〇〇〇〇年〇月〇日」を入れる欄を設ける（日付はプレースホルダーで可）。
- 電話番号・URL・メールアドレスは、【】で囲んだプレースホルダーが残っている場合は、運営側で後から差し替えできるようコメントまたはわかりやすいプレースホルダー表記にしておくこと。

【技術要件】
- HTMLで出力する場合、単体のHTMLファイルで完結するか、もしくはLPの1セクションとして組み込み可能な形式でよい。
- 表は `<table>` または セマンティックなHTML（見出し＋定義リスト等）のいずれかで、アクセシビリティに配慮すること。
```
