# スクール概要 ── AIで少人数で事業を回す実践コース

## コンセプト

### メインコンセプト
**「AI従業員を増やして、少人数で事業を回す」**

抽象的な「AIで分身を作る」ではなく、**具体的に「役割別のAI従業員を構築して、実務で回す」**ことにフォーカスしたスクール。

### なぜこのコンセプトなのか
- 5DAYS講座で「OpenClawでここまでできる」を体感した参加者が、「もっと深く学びたい」と思う自然な延長線
- 「AI分身」は抽象的すぎる → 「AI従業員」は具体的でビジネス層に刺さる
- 実際にいちが自分の事業で実践しており、再現性がある

### ターゲット
- 5DAYS講座に参加した人（最も確度が高い層）
- 起業家・個人事業主・中小企業経営者
  - 人を雇うコストを削減したい
  - AIで業務効率を上げたい
  - 少人数で事業をスケールさせたい
- 副業・複業をしている会社員
  - 本業以外の時間でAIに働いてもらいたい

## スクールの差別化ポイント

### 既存のAIスクールとの違い
| | 他のAIスクール | AirCleスクール |
| --- | --- | --- |
| 教えること | AIツールの使い方 | **AI従業員の作り方＋運用** |
| ゴール | ツールが使えるようになる | **事業の一部がAIで回るようになる** |
| アプローチ | 汎用的 | **あなたの事業に特化** |
| サポート | 動画教材＋質問対応 | **個別ハンズオン＋実装サポート** |
| 講師の実績 | AI評論家が多い | **自ら事業で実践している** |

### 3つの強み
1. **実践ベース**: いちが自分の事業で実際に使っているシステムをそのまま教える
2. **個別カスタマイズ**: 受講者の事業内容に合わせたAI従業員を一緒に構築
3. **コミュニティ**: 受講生同士のナレッジ共有・相互サポート

## 運営形式

### 基本情報
- **形式**: 週1回オンラインセミナー × 8回以上（20:00-22:00）
- **期間**: 8週間
- **プラットフォーム**: Discord（ロール付与でアクセス管理）
- **宿題**: 毎週課題あり
- **ユニコスクール**: 全プランに付帯（AIエージェント数に応じたプラン設計）

### 付き合う前（集客）vs 付き合った後（コンテンツ）の考え方
- **付き合う前（5DAYS講座〜申込）**: キャッチーな訴求、即効性のあるベネフィット提示、X運用等で間口を広く
- **付き合った後（スクール内）**: 本質的な内容を提供、入金後はディスコード即招待、半分くらいの参加率で価値を出す

## カリキュラム

### フェーズ1: メモリ構築（Week 1-2）
- Obsidianを使った「第二の脳」の構築
- マークダウンでの知識整理（宿題：授業内容のマークダウン作成）
- OpenClawのメモリ機能の実装（自分のキャラクター・判断基準を構築）
- セキュリティ対策（Mac mini隔離環境、ネットワーク仮想化）
- APIキーの取得・管理

### フェーズ2: OpenClaw自動化（Week 3-4）
- OpenClaw環境構築
- 基本的な自動化の設定
- GitHub Actionsでデータ自動収集
- チャットベースでの業務自動化（領収書、日報、メール返信等）
- Cursor / Claude Codeとの使い分け

### フェーズ3: マーケティング活用（Week 5-6）
- X運用の自動化（バズ投稿分析 → 自動生成）
- 記事作成の効率化（メモリを活用した高品質コンテンツ）
- 集客導線の設計（X → セミナー → サービス）
- コンテンツビジネスへの応用

### フェーズ4: AI従業員構築（Week 7-8）
- 役割別AI従業員の設計（営業・リサーチ・経理・広報）
- 複数OpenClawの同時運用
- 情報の一元管理（Obsidian × GitHub × OpenClaw）のアーキテクチャ完成
- 承認フローの設計
- 成果報告・ケーススタディ共有
- 卒業後のサポート体制

## 料金設計

### 確定事項（ユニコ氏との会話より）
- **総額**: 税抜 98,000円（ユニコスクール分の9,000円引き後の価格）
- **ユニコスクール付帯**: 全プランにユニコ氏のAIエージェント開発スクールが付帯（実質無料）
- **決済方法**: Upstory × ユニバペイ（ユニバペイ審査通過後に有効化）
- **セミナー内で料金発表**: 第5回セミナーの最後20分で発表（事前にWebでは非公開）

### プラン設計（検討中）
- AIエージェント数に応じたプラン設計を検討
- 全プランにユニコスクールが付帯

### 料金発表の流れ（第5回セミナー内）
1. いちが料金説明を行う
2. ユニコ氏のティーアップ（いちから）
3. ユニコ氏による10分間ピッチ
4. 「一回の冷やかしはやめてください」のスタンスで本気の人だけを集める
5. 個別面談でクロージング（セミナーでは料金と申込方法まで）

## インフラ

### 決済
- **メイン**: Upstory × ユニバペイ
  - ユニバペイの審査通過が必要（特商法表記サイト + 申請フォームの2つを提出済み）
  - 審査通過後、Upstoryと連携して決済を有効化
  - Upstoryにはディスコード連携機能あり（購入者にロールを自動付与）
- **注意**: ユニバペイ審査が通らない場合は代替の決済手段を検討

### コミュニティ
- **Discord**サーバー
  - ロール付与でアクセス管理（Upstory連携で自動化）
  - 受講生専用チャンネル
  - 質問・サポートチャンネル
  - 成果共有チャンネル
  - 週次セミナーの案内チャンネル

### LP
- Upstoryで作成（決済機能内蔵）
- Upstoryの管理画面へのアクセスは確保済み
