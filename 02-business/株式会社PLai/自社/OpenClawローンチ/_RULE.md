# OpenClawローンチ 事業ルール

OpenClawに関するセミナー・スクール・教育事業。
AirCleの活動から派生したローンチ事業で、無料セミナー→有料スクールへの導線を構築。

## 構成

| フォルダ | 内容 |
|----------|------|
| `00_Skills/` | ローンチ運用ガイド・テンプレート |
| `01_無料セミナー_OpenClaw5DAYS講座/` | 全5回の無料セミナー（告知・スライド・MTG・分析） |
| `02_有料スクール_8週間ワークショップ/` | 有料スクール（LINE配信・Discord・面談・LP） |
| `03_スクール設計_商品・法務/` | 商品設計・特商法表記 |
| `04_決済・インフラ/` | 決済設定メモ |

## 主担当C-Suite

- **CMO**: 告知（X投稿・メルマガ・オプチャ・LINE・Peatix）、LP制作
- **CSO**: 個別相談・審査面談の営業
- **CTO**: LP・決済インフラの技術実装
- **CFO**: 売上・決済管理

## ファネル構造

```
X告知 / メルマガ / オプチャ / 公式LINE
    ↓
Peatixで無料セミナー申込
    ↓
5DAYS講座（全5回）
    ↓
審査面談（個別相談）
    ↓
8週間有料ワークショップ
```
