# A シリーズ（無料5DAYS〜審査面談申込まで）の置き場

有料スクールローンチ後は **毎回（第N回）**、無料5DAYSの進行に合わせて公式LINEの文案を差し替える。

## フォルダの意味

| 置き場 | 何を入れるか |
| --- | --- |
| `00_全回共通/` | **A-2〜A-6**（セミナー最終日の翌日以降の追い配信。訴求の骨子は毎回ほぼ同じでよい） |
| `第N回_◯◯/` | **その回だけ変えるもの** … **開催中 Day3・Day4・Day5** の「有料スクール詳細」説明用LINE ＋ **A-1**（最終日当日 or 翌朝のお礼〜面談案内） |
| `_テンプレ_新規回追加用/` | 次の第N回を始めるときに複製するひな形 |

## 運用フロー（毎ローンチ）

1. `_テンプレ_新規回追加用` を `第N回_（無料側フォルダ名と揃える）` にコピーする。
2. **Day3〜Day5** … 当日の講義・スライドと整合するよう、有料ワークショップの詳細訴求を書く（ステップ配信ツールの **Day3/4/5** に貼る想定）。
3. **A-1** … その回のクロージング内容に合わせて書き換える（ユニコデモ回などは別バージョン）。
4. **A-2〜A-6** … 原則 `00_全回共通` を参照。数値・URL・強い言い回しだけ都度メンテ。

B・C は隣フォルダ `03_` `04_` を参照。
