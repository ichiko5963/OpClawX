# 8週間 OpenClawワークショップ ── 個別相談 営業資料

5DAYS講座から申し込んだ方への**個別相談（面談）**で使う営業資料。  
クロージングまで一貫して使えるトーク・流れ・反論処理をまとめた完全版。

---

## 1. 個別相談の目的・ゴール

| 項目 | 内容 |
| --- | --- |
| **目的** | ワークショップの価値を伝え、申込の意思決定をサポートする |
| **ゴール** | ・「自分に合っているか」を本人が判断できる状態にする<br>・申込する場合は、不安なく決済まで進める<br>・申込しない場合は、理由を明確にして気持ちよく終える |
| **やってはいけないこと** | ・一回の冷やかし来店を許さない（本気の人だけが来る導線を維持）<br>・押し売り・長時間の説得 |

---

## 2. 事前準備（面談前チェック）

### 相手の情報を確認する
- 5DAYSのどの回に参加したか（第何回・テーマ）
- 申込時アンケートやメールの内容（興味・課題・事業内容）
- カレンダー登録メモ（希望日時・連絡手段）

### 手元に用意するもの
- 本営業資料（このファイル）
- スライド案に基づいた実際のスライド（または共有画面用PDF）
- 料金・支払方法・特商法の要点メモ
- LPまたは申込ページのURL
- ユニコスクール付帯の説明（「実質無料で付いてくる」）

### 8週間カリキュラム（大まか）

面談で「8週間で何をするか」を伝えるときの概要。

| Week | テーマ | 内容 |
| --- | --- | --- |
| **Week1** | OpenClawの環境構築 | 使えるようになろう |
| **Week2** | メモリ構築をしよう！ | ObsidianとCursor |
| **Week3** | メモリの環境構築！（OpenClaw編） | Memory Systems作成 |
| **Week4** | 【基礎〜応用】知識インプットDAY | GitHubやVercelの基礎／開発をできるように／連携をできるように |
| **Week5** | 情報の一元管理 | GitHub Actionsでデータ自動収集 |
| **Week6** | コンテンツビジネスへのマーケティング活用フル講義 | X・note・メルマガ・LP作成 |
| **Week7** | チャットで開発！自分の事業の業務自動化 | 領収書・日報・メール返信等 |
| **Week8** | 役割別AI従業員の設計＋成果報告 | 営業・リサーチ・経理・マーケ／OpenClawで作ったスライド構成で成果を共有 |

## 4. 話すポイント（パート別）

### オープニング
- 「今日は30分程度で、ワークショップの中身と、ご自身に合うかどうかをお話しします。最後に申し込み方法だけお伝えするので、今日中に決めていただかなくても大丈夫です」  
  → ただし、**「一回だけの冷やかし来店はご遠慮ください」**は、5DAYSや申込時にはっきり伝えている前提で、面談では柔らかく「本気で検討されている方だけお時間いただいている」程度に留めても可。
- カメラ・音声の確認（オンラインの場合）。

### ヒアリング
- 「5DAYSのどこが一番刺さりましたか？」
- 「今、OpenClawやAIで一番実現したいことは何ですか？」
- 「事業や仕事で、人を増やさずに回したい部分はありますか？」
- メモを取る（後で「〇〇さんの場合は、Week5〜6のマーケ活用が特に役立ちそうです」のように紐づける）。

### ワークショップの全体像
- **コンセプト**：「AI従業員を増やして、少人数で事業を回す」。
- **4ステップ**：  
  Step1 メモリ構築（Week1〜2）→ Step2 OpenClaw自動化（Week3〜4）→ Step3 マーケ活用（Week5〜6）→ Step4 AI従業員構築（Week7〜8）。
- **差別化**：いちが自分の事業で実際に使っているシステムをそのまま教える・受講者の事業に合わせた個別カスタマイズ・Discordで質問し放題。

### カリキュラム・付帯
- 週1回・20:00〜22:00・8週間。毎週宿題あり。Discordで質問・サポート。
- **全プランにユニコ氏のAIエージェント開発スクールが付帯（実質無料）**。ここを明確に伝える。

### 料金・支払・申込
- **税込 107,800円**（税抜 98,000円。ユニコスクール分9,000円引き後の価格）。
- 決済：ユニバペイ経由（クレジットカード）。審査制のため、申込フォームから申請→審査通過後に決済リンク案内。
- 申込方法：LP（またはUpstory）の申込フォームから。決済完了後、24時間以内にDiscord招待等を案内。

### クロージング
- **申込する場合**：「では、〇〇のリンクからお申し込みください。審査通過後に決済のご案内を送ります。決済完了後、24時間以内にDiscordの招待をお送りします。」
- **迷っている場合**：「今日はここまでにします。ご不明点があれば、メールでいつでも聞いてください。申込は〇〇まで受け付けています。」
- **申込しない場合**：「無理にとは言いません。また機会があれば、5DAYSや別のイベントでお会いしましょう。」

## 6. 注意事項（営業側）

- **特商法**：料金・支払方法・キャンセル・返金条件は、特商法表記および契約内容と矛盾しないように話す。
- **審査制**：申込は審査制であることを伝え、冷やかしや一回落ちる前提の来店を防ぐ導線であることを説明してもよい。
- **一回の冷やかし来店禁止**：5DAYSやメールで「本気で検討されている方のみ」と明記している前提を崩さない。
