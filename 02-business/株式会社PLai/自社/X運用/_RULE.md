# X運用 事業ルール

## 運用アカウント

### AirCle (@aircle_ai)
- 日本最大級の学生AI団体
- パス: `AirCle/`
- バイブコーディング講義、note配布、ワークショップ
- **バズパターン**: 【速報】系、【無料配布】系、150〜300文字のノート投稿

### いち@OpenClawガチ勢 (@ichiaimarketer)
- AI×マーケティングの個人ブランド
- パス: `いち-OpenClawガチ勢/`
- AIツール分析、マーケ戦略、バイブコーディング

## 各アカウント共通の構成

```
[アカウント名]/
├── _RULE.md         ← アカウント固有ルール
├── overview.md      ← アカウント概要・フォロワー数・目標
├── 口調プロンプト.md ← そのアカウントのトーン定義
├── 投稿/            ← 作成した投稿（日付順）
├── 記事/            ← note等の記事
├── 過去投稿csv/      ← 過去の投稿データ（分析用）
├── mtg/             ← 運用に関するMTG
└── projects/        ← 金銭発生の特定キャンペーン等
```

## 投稿作成時の参照順序

1. `口調プロンプト.md` — トーンを合わせる
2. `過去投稿csv/` — 過去のバズパターンを確認
3. `c-suite/cmo/skills/X投稿テンプレート50個.md` — テンプレ参照
4. `c-suite/cmo/skills/X投稿具体例50個.md` — 具体例参照
5. `skills/x-bazu-prompt/SKILL.md` — バズ投稿生成スキル
