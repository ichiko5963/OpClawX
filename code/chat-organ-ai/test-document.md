# RAGシステムテスト用ドキュメント

## 概要
このドキュメントは、RAG（Retrieval Augmented Generation）システムのテスト用に作成されました。

## 機能
- ファイルのチャンキング
- ベクトル化
- 意味ベースの検索

## 技術スタック
- Mastra Framework
- OpenAI Embeddings
- TypeScript
- Next.js

## 使用方法
1. ファイルをアップロード
2. チャンキングとベクトル化
3. 検索クエリの実行
4. 結果の表示

## 注意事項
- ファイルサイズの制限
- API制限の考慮
- エラーハンドリングの実装
