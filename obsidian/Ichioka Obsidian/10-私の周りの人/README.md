# 私の周りの人

このフォルダには、いちさんが関わる人々の情報をまとめています。

## 構造
```
10-私の周りの人/
├── [人物名]/
│   ├── PROFILE.md      # 基本情報（名前、連絡先、関係性など）
│   ├── MTG_YYYY-MM-DD_タイトル.md  # ミーティング議事録
│   └── NOTES.md        # その他メモ
```

## 使い方
- **新しい人を追加**: Piにミーティング議事録や人の情報を渡すと、自動でフォルダを作成
- **情報を検索**: Piに「〇〇さんについて教えて」と聞くと、その人のフォルダから情報を取得
