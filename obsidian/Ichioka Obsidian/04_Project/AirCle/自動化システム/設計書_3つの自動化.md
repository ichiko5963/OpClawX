# 1. ピーティックス購入者メール案生成

## 要件
- イベントは複数ある
- YouTubeリンク + noteURL を購入者に送る
- りょうせいがアップロード → いちさんがOK → Peatixから手動送信
- **Piがやること**: メール文案の生成のみ

## フロー
```
りょうせい: YouTube限定公開リンクを共有
    ↓
いちさん: 「OK、これで行こう」
    ↓
Pi: イベント名・YouTubeリンク・noteURLを受け取る
    ↓
Pi: メールテンプレートに当てはめて文案生成
    ↓
いちさん: Peatixから手動送信
```

# 2. 講義リマインダー自動送信

## 要件
- Googleカレンダーに講義予定が入っている
- Slackの「4番の勉強会・交流会」チャンネルに送信
- 3日前・1日前・12時間前にリマインド
- 運営全員向け

## フロー
```
Googleカレンダーにイベント登録
    ↓
n8n/Zapier が定期チェック（毎時）
    ↓
3日前/1日前/12時間前 を検知
    ↓
Slackにリマインドメッセージ送信
```

## 技術構成

### 選択肢A: n8n（推奨）
- 無料・セルフホスト可能
- Googleカレンダー連携あり
- Slack連携あり

### 選択肢B: Zapier
- 有料（月額）
- 設定が簡単

### 選択肢C: Google Apps Script
- 無料
- カスタマイズ自由
- 少しコーディング必要

## 実装手順（n8n想定）
1. n8nをセットアップ（Vercelまたはローカル）
2. Googleカレンダートリガー設定
3. 日付計算ノード追加
4. Slack送信ノード追加
5. テスト実行

## 所要時間
- n8n: 2-3時間
- GAS: 1-2時間

# 3. 請求書管理システム（LINE公式連携）

## 要件
- LINE公式アカウント新規作成
- 複数人から請求書が来る（りょうせい、他）
- 画像(JPEG)またはPDFで送信
- OCRでテキスト化
- Googleスプレッドシートに自動記録
- チェックボックスで承認 → 振込通知自動送信

## 全体フロー
```
運営メンバー: LINE公式に請求書画像/PDF送信
    ↓
LINE Messaging API: Webhook受信
    ↓
n8n/GAS: 画像/PDFを取得
    ↓
OCR（Google Vision API）: テキスト抽出
    ↓
AI解析: 金額・項目・送信者を構造化
    ↓
Googleスプレッドシート: 新規行追加
    ↓
いちさん: スプシでチェックボックスON
    ↓
GAS: チェック検知 → LINE通知送信
    ↓
送信者: 「承認されました」通知受信
```

## システム構成

### 1. LINE公式アカウント
- LINE Official Account Manager で作成
- Messaging API を有効化
- Webhook URL を設定

### 2. バックエンド（n8n または GAS）
- Webhook受信
- 画像/PDF取得
- OCR処理
- スプシ書き込み

### 3. OCR
- **Google Cloud Vision API**（推奨）
  - 日本語対応
  - 高精度
  - 月1000件まで無料

### 4. AI解析
- OCR結果から以下を抽出：
  - 送信者名
  - 請求金額
  - 内訳（あれば）
  - 振込先（あれば）

### 5. Googleスプレッドシート

| 受信日時 | 送信者 | 金額 | 内訳 | 源泉徴収後 | 画像URL | 承認 | 通知済 |
|----------|--------|------|------|------------|---------|------|--------|
| 2026-02-04 | りょうせい | 12,000 | 動画編集3本 | 10,775 | https://... | ☐ | - |

- **源泉徴収後**: 自動計算（金額 × 0.8979）
- **承認**: チェックボックス
- **通知済**: 通知送信後に自動で✓

### 6. 承認→通知フロー

**GAS（Google Apps Script）で実装:**
```
onEdit トリガー
    ↓
「承認」列がTRUEに変更されたか検知
    ↓
まだ「通知済」でなければ
    ↓
LINE Messaging API で送信者に通知
    ↓
「通知済」を✓に更新
```

**通知メッセージ案:**
```
✅ 請求書が承認されました

━━━━━━━━━━━━━━━━━
金額: {金額}円
源泉徴収後: {源泉徴収後}円
━━━━━━━━━━━━━━━━━

振込は月末に行います。
ありがとうございました！
```

## 必要なもの

### いちさん側で準備
1. **LINE Official Account** 作成
   - https://manager.line.biz/ からアカウント作成
   - Messaging API を有効化
   - Channel Access Token を取得
   - Webhook URL を設定（後で共有）

2. **Google Cloud Platform** アカウント（Vision API用）
   - https://console.cloud.google.com/
   - Vision API を有効化
   - サービスアカウントキー（JSON）を取得

3. **Googleスプレッドシート** 作成
   - テンプレートはPiが作成

### Pi側で構築
1. n8n ワークフロー または GAS スクリプト
2. OCR連携
3. スプレッドシート連携
4. LINE通知連携

## セキュリティ考慮
- 請求書画像は一定期間後に削除
- スプレッドシートのアクセス権限を限定
- LINE Channel Access Token は環境変数で管理

## 所要時間
- LINE公式アカウント設定: 30分（いちさん）
- GCP設定: 30分（いちさん or Pi）
- バックエンド構築: 4-5時間（Pi）
- テスト・調整: 1-2時間
