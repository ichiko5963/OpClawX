# 連携企業

このフォルダには、いちさん/AirCleが連携する企業・組織の情報をまとめています。

## 構造
```
11-連携企業/
├── [企業名]/
│   ├── PROFILE.md      # 企業情報（概要、担当者、関係性など）
│   ├── MTG_YYYY-MM-DD_タイトル.md  # ミーティング議事録
│   └── NOTES.md        # その他メモ
```

## 使い方
- **新しい企業を追加**: Piにミーティング議事録を渡すと、自動でフォルダを作成
- **情報を検索**: Piに「〇〇社について教えて」と聞くと、その企業のフォルダから情報を取得
