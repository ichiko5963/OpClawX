# 10_People

人物プロファイルを格納。

## 用途
- いちさんの周りにいる人の情報
- 連絡先・関係性
- 過去のやり取りメモ

## ファイル構造
各人物のフォルダ内に `PROFILE.md` を作成し、以下を記録：
- 基本情報（名前、連絡先）
- どういう関係か
- 過去のMTG・やり取り
- 備考
