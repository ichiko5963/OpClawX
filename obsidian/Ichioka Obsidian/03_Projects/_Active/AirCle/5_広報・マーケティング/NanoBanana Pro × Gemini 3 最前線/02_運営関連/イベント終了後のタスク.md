# イベント終了後のタスク

## 即座に実行するタスク

### セミナー終了直後
- [ ] 参加者への感謝メッセージ送信
- [ ] Xでの終了報告投稿
- [ ] オープンチャットでの終了感謝メッセージ

### 1〜2日後
- [ ] 特典配布（参加者全員）
- [ ] アーカイブ動画編集開始
- [ ] 参加者アンケートの準備

### 2〜3日後
- [ ] アーカイブ動画配布（有料チケット購入者）
- [ ] 参加者アンケートの実施
- [ ] 実績アーカイブの記録

## 1週間以内に実行するタスク

- [ ] 参加者アンケートの集計
- [ ] 次回改善点の整理
- [ ] 実績アーカイブの完成
- [ ] 次回以降のアクションプランの作成

## 1ヶ月以内に実行するタスク

- [ ] 分析レポートの作成
- [ ] 次回セミナーへの反映事項の整理
- [ ] コミュニティへの報告
