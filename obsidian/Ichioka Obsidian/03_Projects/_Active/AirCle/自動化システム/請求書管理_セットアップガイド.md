# AirCle 請求書管理システム セットアップガイド

## 概要

LINE公式アカウントに請求書画像を送信すると、自動でGoogleスプレッドシートに記録され、
承認チェックを入れると送信者に通知が届くシステム。

---

## システム構成

```
運営メンバー → LINE公式に画像送信
                    ↓
            Google Apps Script (Webhook)
                    ↓
            画像保存 + OCR解析
                    ↓
            スプレッドシートに記録
                    ↓
            いちさんが承認チェック
                    ↓
            送信者にLINE通知
```

---

## セットアップ手順

### Step 1: Googleスプレッドシート作成

1. 新しいスプレッドシートを作成
2. シート名を「請求書」に変更
3. 1行目（ヘッダー）に以下を入力:

| A | B | C | D | E | F | G | H | I | J |
|---|---|---|---|---|---|---|---|---|---|
| 受信日時 | LINE名 | LINE ID | 金額 | 内訳 | 源泉徴収後 | 画像URL | 承認 | 通知済 | メモ |

4. H列（承認）はチェックボックス形式に設定
   - H2以降を選択 → 挿入 → チェックボックス

5. F列（源泉徴収後）に数式を設定
   - F2: `=IF(D2="","",ROUND(D2*0.8979,0))`
   - 下にコピー

### Step 2: Google Apps Script 設定

1. スプレッドシートで「拡張機能」→「Apps Script」
2. 以下のコードを貼り付け
3. `CHANNEL_ACCESS_TOKEN` と `CHANNEL_SECRET` を設定
4. 保存

### Step 3: Webアプリとしてデプロイ

1. 「デプロイ」→「新しいデプロイ」
2. 種類: 「ウェブアプリ」
3. 実行ユーザー: 「自分」
4. アクセスできるユーザー: 「全員」
5. デプロイ → URLをコピー

### Step 4: LINE Webhook設定

1. LINE Developers Console で Webhook URL を設定
2. Step 3 で取得したURLを貼り付け
3. Webhook を有効化
4. 「検証」ボタンで接続テスト

---

## 使い方

### 請求書を送る人（運営メンバー）

1. LINE公式アカウントを友だち追加
2. 請求書の画像を送信
3. 続けてテキストで「12000円 動画編集3本」のように金額と内訳を送信
4. 承認されるとLINEで通知が届く

### 承認する人（いちさん）

1. スプレッドシートを開く
2. 新しい請求書が追加されている
3. 画像URLをクリックして内容確認
4. 問題なければ「承認」にチェック
5. 自動で送信者に通知が届く

---

## トラブルシューティング

### 画像が届かない
- LINE Webhook URLが正しく設定されているか確認
- GASのログを確認（実行 → 実行ログ）

### 通知が届かない
- Channel Access Token が正しいか確認
- 送信者のLINE IDがスプレッドシートに記録されているか確認
