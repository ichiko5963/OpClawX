# AirCle 請求書管理システム - 完全セットアップ手順

## 🎯 完成イメージ

```
運営メンバー: 請求書画像をLINEで送信
    ↓
Bot: 「金額と内訳を送ってください」
    ↓
運営メンバー: 「12000円 動画編集3本」
    ↓
Bot: 「受け付けました！承認されるとお知らせします」
    ↓
スプシに自動記録（源泉徴収も自動計算）
    ↓
いちさん: チェックボックスで承認
    ↓
運営メンバーに自動通知: 「承認されました！振込は月末です」
```

---

## Step 1: LINE Official Account 作成（5分）

### 1.1 アカウント作成

1. https://manager.line.biz/ にアクセス
2. LINEアカウントでログイン
3. 「アカウントを作成」をクリック
4. 以下を入力:
   - **アカウント名**: `AirCle経費管理`
   - **メールアドレス**: いちさんのメール
   - **業種**: 「個人（その他）」でOK
   - **運用目的**: 適当でOK
5. 「確認」→「完了」

### 1.2 Messaging API 有効化

1. 作成したアカウントの管理画面を開く
2. 右上の「設定」をクリック
3. 左メニューの「Messaging API」をクリック
4. 「Messaging APIを利用する」をクリック
5. プロバイダー:
   - 既存があれば選択
   - なければ「プロバイダーを作成」→ 名前: `AirCle`
6. 「同意する」にチェック → 「OK」

### 1.3 Channel Access Token 取得

1. LINE Developers Console が開く（または https://developers.line.biz/console/ ）
2. 作成したチャンネルをクリック
3. 「Messaging API設定」タブ
4. 下にスクロール → 「チャネルアクセストークン（長期）」
5. 「発行」をクリック
6. **表示されたトークンをコピーして保存**（後で使う）

### 1.4 Channel Secret 取得

1. 「チャネル基本設定」タブ
2. 「チャネルシークレット」をコピーして保存

---

## Step 2: Googleスプレッドシート作成（3分）

### 2.1 新規作成

1. https://sheets.google.com/ にアクセス
2. 「空白」で新規作成
3. ファイル名: `AirCle_請求書管理`

### 2.2 ヘッダー設定

1行目に以下を入力（A1〜J1）:

| A | B | C | D | E | F | G | H | I | J |
|---|---|---|---|---|---|---|---|---|---|
| 受信日時 | LINE名 | LINE ID | 金額 | 内訳 | 源泉徴収後 | 画像URL | 承認 | 通知済 | メモ |

### 2.3 チェックボックス設定

1. H2:H1000 を選択
2. メニュー「挿入」→「チェックボックス」
3. I2:I1000 も同様にチェックボックスに

### 2.4 源泉徴収の自動計算

1. F2 に入力: `=IF(D2="","",ROUND(D2*0.8979,0))`
2. F2 を選択 → Ctrl+C
3. F3:F1000 を選択 → Ctrl+V

---

## Step 3: Google Apps Script 設定（5分）

### 3.1 スクリプトエディタを開く

1. スプレッドシートで「拡張機能」→「Apps Script」
2. 新しいタブでスクリプトエディタが開く

### 3.2 コードを貼り付け

1. デフォルトの `function myFunction() {}` を全て削除
2. `請求書管理_GAS.js` の内容を全て貼り付け

### 3.3 設定を編集

コードの上部にある `CONFIG` を編集:

```javascript
const CONFIG = {
  CHANNEL_ACCESS_TOKEN: 'ここにStep1.3で取得したトークンを貼る',
  CHANNEL_SECRET: 'ここにStep1.4で取得したシークレットを貼る',
  DRIVE_FOLDER_ID: '',  // 空のままでOK（自動作成される）
  SPREADSHEET_ID: '',   // 空のままでOK（このシートを使う）
  SHEET_NAME: '請求書'  // シート名を変えた場合は修正
};
```

### 3.4 保存

1. Ctrl+S または 💾アイコンをクリック
2. プロジェクト名を入力（例: `AirCle請求書管理`）

---

## Step 4: Webアプリとしてデプロイ（3分）

### 4.1 デプロイ

1. 右上の「デプロイ」→「新しいデプロイ」
2. ⚙️アイコン → 「ウェブアプリ」を選択
3. 設定:
   - **説明**: `AirCle請求書管理システム`
   - **実行ユーザー**: `自分`
   - **アクセスできるユーザー**: `全員`
4. 「デプロイ」をクリック

### 4.2 権限承認

1. 「アクセスを承認」をクリック
2. Googleアカウントを選択
3. 「詳細」→「〇〇（安全ではないページ）に移動」
4. 「許可」をクリック

### 4.3 URLをコピー

1. 「ウェブアプリ」の下に表示されるURLをコピー
2. **このURLを保存**（次のステップで使う）

例: `https://script.google.com/macros/s/xxxxxxxxxxxxx/exec`

---

## Step 5: LINE Webhook設定（2分）

### 5.1 Webhook URL 設定

1. LINE Developers Console に戻る
2. 「Messaging API設定」タブ
3. 「Webhook URL」の「編集」をクリック
4. Step 4.3 でコピーしたURLを貼り付け
5. 「更新」をクリック

### 5.2 Webhook 有効化

1. 「Webhookの利用」を **オン** にする

### 5.3 接続テスト

1. 「検証」ボタンをクリック
2. 「成功」と表示されればOK！

### 5.4 その他の設定

1. 「応答メッセージ」を **オフ** にする
2. 「あいさつメッセージ」はお好みで（オフ推奨）

---

## Step 6: onEdit トリガー設定（2分）

承認時の自動通知を有効にする。

### 6.1 トリガー設定

1. Apps Script エディタに戻る
2. 左メニューの ⏰「トリガー」をクリック
3. 右下の「+ トリガーを追加」

### 6.2 設定内容

- **実行する関数**: `onEdit`
- **イベントのソース**: `スプレッドシートから`
- **イベントの種類**: `編集時`

3. 「保存」をクリック

---

## Step 7: テスト（2分）

### 7.1 友だち追加

1. LINE Official Account Manager で QRコード を表示
2. 自分のLINEで友だち追加

### 7.2 動作確認

1. 適当な画像を送信
2. 「金額と内訳を送ってください」と返信が来る
3. 「1000円 テスト」と送信
4. 「受け付けました」と返信が来る
5. スプレッドシートに記録されているか確認
6. 承認チェックを入れる
7. LINE に通知が届く

## トラブルシューティング

### 「検証」でエラーが出る

- Webhook URL が正しくコピーされているか確認
- デプロイが完了しているか確認
- 「アクセスできるユーザー」が「全員」になっているか確認

### 画像を送っても反応がない

- Apps Script のログを確認（実行 → 実行ログ）
- CHANNEL_ACCESS_TOKEN が正しいか確認

### 承認しても通知が届かない

- onEdit トリガーが設定されているか確認
- LINE ID（C列）が正しく記録されているか確認

### 権限エラーが出る

- Google Drive へのアクセス権限を再承認
- Apps Script を再デプロイ
