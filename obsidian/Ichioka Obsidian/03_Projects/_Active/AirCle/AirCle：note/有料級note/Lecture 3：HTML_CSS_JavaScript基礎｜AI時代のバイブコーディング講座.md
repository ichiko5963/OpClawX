### SECTION 01: HTML5の基本とセマンティックタグ（30分）

#### 1.1 HTMLとは？構造を定義する言語

**HTML（HyperText Markup Language）**は、Webページの**構造を定義する言語**です。

**HTMLの役割**:
- ページの**骨組み**を作る
- **見出し、段落、リスト、画像**などの要素を配置
- **セマンティック（意味のある）タグ**で構造を明確にする

**基本的なHTMLの構造**:
```html
<!DOCTYPE html>
<html lang="ja">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>ページのタイトル</title>
</head>
<body>
    <!-- ここにコンテンツを書く -->
</body>
</html>
```

#### 1.2 主要タグ: div, section, article, header, footer, nav, aside

**セマンティックタグ**は、**意味を持ったタグ**です。これらを使うことで、検索エンジンやスクリーンリーダーがページの構造を理解しやすくなります。

**主要なセマンティックタグ**:

| タグ | 意味 | 使用例 |
|------|------|--------|
| **`<header>`** | ページのヘッダー | ロゴ、ナビゲーション |
| **`<nav>`** | ナビゲーション | メニュー、リンク集 |
| **`<main>`** | メインコンテンツ | 記事本文 |
| **`<section>`** | セクション | 関連するコンテンツのグループ |
| **`<article>`** | 記事 | 独立したコンテンツ |
| **`<aside>`** | サイドバー | 補足情報、広告 |
| **`<footer>`** | フッター | 著作権情報、リンク |

**実例: セマンティックな構造**:
```html
<!DOCTYPE html>
<html lang="ja">
<head>
    <meta charset="UTF-8">
    <title>ブログサイト</title>
</head>
<body>
    <header>
        <h1>私のブログ</h1>
        <nav>
            <ul>
                <li><a href="#home">ホーム</a></li>
                <li><a href="#about">について</a></li>
                <li><a href="#contact">お問い合わせ</a></li>
            </ul>
        </nav>
    </header>
    
    <main>
        <article>
            <h2>記事タイトル</h2>
            <p>記事の本文...</p>
        </article>
    </main>
    
    <aside>
        <h3>関連記事</h3>
        <ul>
            <li>記事1</li>
            <li>記事2</li>
        </ul>
    </aside>
    
    <footer>
        <p>&copy; 2025 私のブログ</p>
    </footer>
</body>
</html>
```

#### 1.3 フォーム要素: input, textarea, select, button

**フォーム要素**は、ユーザーからの**入力を受け取る要素**です。

**主要なフォーム要素**:

```html
<!-- テキスト入力 -->
<input type="text" placeholder="名前を入力">
<input type="email" placeholder="メールアドレス">
<input type="password" placeholder="パスワード">

<!-- テキストエリア（複数行） -->
<textarea placeholder="メッセージを入力"></textarea>

<!-- セレクトボックス -->
<select>
    <option>選択してください</option>
    <option>オプション1</option>
    <option>オプション2</option>
</select>

<!-- ボタン -->
<button type="submit">送信</button>
```

**実践: セマンティックな構造のブログページ**

**課題**: セマンティックタグを使って、ブログページの構造を作成

**手順**:
1. `blog.html`を作成
2. header、nav、main、article、aside、footerを使う
3. 適切な見出しタグ（h1, h2, h3）を使う

### SECTION 02: CSSの基礎とFlexbox（35分）

#### 2.1 CSSとは？スタイルを定義する言語

**CSS（Cascading Style Sheets）**は、Webページの**見た目を定義する言語**です。

**CSSの役割**:
- **色、サイズ、レイアウト**を指定
- **レスポンシブデザイン**（画面サイズに応じた表示）を実現
- **アニメーション**や**トランジション**を追加

**CSSの基本的な書き方**:
```css
/* セレクタ { プロパティ: 値; } */
h1 {
    color: blue;
    font-size: 24px;
}
```

#### 2.2 セレクタ: class, id, 子孫セレクタ、疑似クラス

**セレクタ**は、**どの要素にスタイルを適用するか**を指定します。

**主要なセレクタ**:

```css
/* 要素セレクタ */
p { color: red; }

/* クラスセレクタ */
.button { background: blue; }

/* IDセレクタ */
#header { height: 100px; }

/* 子孫セレクタ */
.container p { color: green; }

/* 疑似クラス */
button:hover { background: darkblue; }
a:active { color: purple; }
```

**HTMLでの使い方**:
```html
<div class="container">
    <p>この段落は緑色</p>
    <button class="button">このボタンは青色</button>
</div>
```

#### 2.3 Flexbox: 横並び・縦並びの自由自在なレイアウト

**Flexbox**は、**柔軟なレイアウト**を作るためのCSS機能です。

**Flexboxの基本**:
```css
.container {
    display: flex;
    flex-direction: row; /* 横並び（デフォルト） */
    justify-content: center; /* 横方向の配置 */
    align-items: center; /* 縦方向の配置 */
}
```

**Flexboxの主要プロパティ**:

| プロパティ | 意味 | 値の例 |
|-----------|------|--------|
| **`display: flex`** | Flexboxを有効化 | - |
| **`flex-direction`** | 方向 | `row`（横）、`column`（縦） |
| **`justify-content`** | 横方向の配置 | `center`、`space-between`、`flex-start` |
| **`align-items`** | 縦方向の配置 | `center`、`flex-start`、`flex-end` |
| **`flex-wrap`** | 折り返し | `wrap`、`nowrap` |

**実践: ヘッダー + カード3列のレイアウト**

**課題**: Flexboxを使って、ヘッダーとカード3列のレイアウトを作成

**手順**:
1. `layout.html`を作成
2. Flexboxでヘッダーを横並びに
3. Flexboxでカードを3列に配置

### SECTION 03: CSS Grid Layoutとレスポンシブデザイン（30分）

#### 3.1 Grid Layout: 複雑なレイアウトを簡単に実現

**CSS Grid**は、**2次元のレイアウト**を作るためのCSS機能です。

**Gridの基本**:
```css
.container {
    display: grid;
    grid-template-columns: 1fr 1fr 1fr; /* 3列 */
    grid-template-rows: 100px 200px; /* 2行 */
    gap: 20px; /* 間隔 */
}
```

**Gridの主要プロパティ**:

| プロパティ | 意味 | 値の例 |
|-----------|------|--------|
| **`display: grid`** | Gridを有効化 | - |
| **`grid-template-columns`** | 列の定義 | `1fr 1fr 1fr`（3等分） |
| **`grid-template-rows`** | 行の定義 | `100px 200px` |
| **`gap`** | 間隔 | `20px` |
| **`grid-column`** | 列の範囲 | `1 / 3`（1列目から3列目まで） |

#### 3.2 メディアクエリ: 画面サイズに応じてスタイルを変更

**メディアクエリ**は、**画面サイズに応じてスタイルを変更**する機能です。

**基本的な書き方**:
```css
/* デフォルト（スマホ） */
.container {
    width: 100%;
}

/* タブレット以上 */
@media (min-width: 768px) {
    .container {
        width: 750px;
        margin: 0 auto;
    }
}

/* PC以上 */
@media (min-width: 1024px) {
    .container {
        width: 1200px;
    }
}
```

**実践: PC/スマホ対応のギャラリーページ**

**課題**: メディアクエリを使って、PCでは3列、スマホでは1列のギャラリーを作成

### SECTION 04: JavaScriptの基礎文法（40分）

#### 4.1 変数: const, let, var

**変数**は、**データを保存する箱**です。

**変数の宣言方法**:
```javascript
// const: 再代入不可（推奨）
const name = "山田太郎";

// let: 再代入可能
let age = 20;
age = 21; // OK

// var: 古い書き方（使わない）
var old = "使わない";
```

#### 4.2 データ型: 文字列、数値、真偽値、配列、オブジェクト

**JavaScriptの主要なデータ型**:

```javascript
// 文字列
const text = "Hello, World!";

// 数値
const number = 42;
const float = 3.14;

// 真偽値
const isTrue = true;
const isFalse = false;

// 配列
const fruits = ["りんご", "バナナ", "オレンジ"];

// オブジェクト
const person = {
    name: "山田太郎",
    age: 20,
    city: "東京"
};
```

#### 4.3 関数: function宣言、アロー関数

**関数**は、**処理をまとめたもの**です。

**関数の書き方**:
```javascript
// function宣言
function greet(name) {
    return `こんにちは、${name}さん！`;
}

// アロー関数（推奨）
const greet2 = (name) => {
    return `こんにちは、${name}さん！`;
};

// 1行で書ける場合
const greet3 = (name) => `こんにちは、${name}さん！`;
```

#### 4.4 条件分岐: if文、三項演算子

**条件分岐**は、**条件に応じて処理を分ける**機能です。

```javascript
// if文
if (age >= 20) {
    console.log("成人です");
} else {
    console.log("未成年です");
}

// 三項演算子
const message = age >= 20 ? "成人です" : "未成年です";
```

#### 4.5 ループ: for, forEach, map

**ループ**は、**繰り返し処理**を実行する機能です。

```javascript
// for文
for (let i = 0; i < 5; i++) {
    console.log(i);
}

// forEach
fruits.forEach((fruit) => {
    console.log(fruit);
});

// map（新しい配列を作る）
const upperFruits = fruits.map((fruit) => fruit.toUpperCase());
```

### SECTION 05: DOM操作とイベントリスナー（35分）

#### 5.1 DOM（Document Object Model）とは？

**DOM**は、**HTMLをJavaScriptで操作するための仕組み**です。

**DOMの基本**:
- HTMLの各要素は**DOM要素**として扱われる
- JavaScriptでDOM要素を**取得・変更・追加・削除**できる

#### 5.2 要素の取得: getElementById, querySelector

**要素の取得方法**:

```javascript
// IDで取得
const header = document.getElementById('header');

// クラスで取得（最初の1つ）
const button = document.querySelector('.button');

// クラスで取得（全て）
const buttons = document.querySelectorAll('.button');

// タグで取得
const paragraphs = document.querySelectorAll('p');
```

#### 5.3 要素の操作: textContent, innerHTML, classList

**要素の操作**:

```javascript
// テキストを変更
element.textContent = "新しいテキスト";

// HTMLを変更
element.innerHTML = "<strong>太字</strong>";

// クラスを追加
element.classList.add('active');

// クラスを削除
element.classList.remove('active');

// クラスをトグル
element.classList.toggle('active');
```

#### 5.4 イベント: click, input, submit, keypress

**イベントリスナー**は、**ユーザーの操作に反応する**機能です。

```javascript
// クリックイベント
button.addEventListener('click', () => {
    alert('クリックされました！');
});

// 入力イベント
input.addEventListener('input', (e) => {
    console.log(e.target.value);
});

// フォーム送信イベント
form.addEventListener('submit', (e) => {
    e.preventDefault(); // デフォルトの動作をキャンセル
    console.log('フォームが送信されました');
});
```

**実践: ボタンクリックでテキストを変更**

**課題**: ボタンをクリックすると、テキストが変更される機能を作成

**ステップ3: JavaScriptで機能を実装**

// テーマ切り替え
const themeToggle = document.getElementById('themeToggle');
const html = document.documentElement;

themeToggle.addEventListener('click', () => {
    const currentTheme = html.getAttribute('data-theme');
    const newTheme = currentTheme === 'dark' ? 'light' : 'dark';
    html.setAttribute('data-theme', newTheme);
    themeToggle.textContent = newTheme === 'dark' ? '☀️' : '🌙';
    saveTheme(newTheme);
});

// テーマを保存
function saveTheme(theme) {
    localStorage.setItem('theme', theme);
}

// テーマを読み込み
function loadTheme() {
    const savedTheme = localStorage.getItem('theme') || 'light';
    html.setAttribute('data-theme', savedTheme);
    themeToggle.textContent = savedTheme === 'dark' ? '☀️' : '🌙';
}

// Todo機能
const todoForm = document.getElementById('todoForm');
const taskInput = document.getElementById('taskInput');
const taskList = document.getElementById('taskList');

// タスクを読み込み
function loadTasks() {
    const tasks = JSON.parse(localStorage.getItem('tasks') || '[]');
    taskList.innerHTML = '';
    tasks.forEach((task, index) => {
        const li = document.createElement('li');
        li.className = `task-item ${task.completed ? 'completed' : ''}`;
        li.innerHTML = `
            <span onclick="toggleTask(${index})">${task.text}</span>
            <button onclick="deleteTask(${index})">削除</button>
        `;
        taskList.appendChild(li);
    });
}

// タスクを追加
todoForm.addEventListener('submit', (e) => {
    e.preventDefault();
    const taskText = taskInput.value.trim();
    if (taskText === '') return;
    
    const tasks = JSON.parse(localStorage.getItem('tasks') || '[]');
    tasks.push({ text: taskText, completed: false });
    localStorage.setItem('tasks', JSON.stringify(tasks));
    
    taskInput.value = '';
    loadTasks();
});

// タスクをトグル
function toggleTask(index) {
    const tasks = JSON.parse(localStorage.getItem('tasks') || '[]');
    tasks[index].completed = !tasks[index].completed;
    localStorage.setItem('tasks', JSON.stringify(tasks));
    loadTasks();
}

// タスクを削除
function deleteTask(index) {
    const tasks = JSON.parse(localStorage.getItem('tasks') || '[]');
    tasks.splice(index, 1);
    localStorage.setItem('tasks', JSON.stringify(tasks));
    loadTasks();
}

// 初期化
loadTheme();
loadTasks();
