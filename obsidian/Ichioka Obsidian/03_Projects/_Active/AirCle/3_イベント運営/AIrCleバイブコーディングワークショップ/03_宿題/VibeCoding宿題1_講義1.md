# 宿題1: 自分だけのサービスを作ろう

## この宿題でやること

講義1で学んだ**GensparkかManus**を使って、**自分だけのサービス**を作ります。

テーマは自由！カフェ以外でもOK。

---

## サービスって何？

講義で説明したけど、おさらい。

| | ホームページ | サービス |
|---|------------|---------|
| 例 | 会社紹介、お店の案内 | 食べログ、メルカリ、Instagram |
| 特徴 | 見るだけ | **使える**、**動く** |

今回作るのは**サービス**のほう。「使えるもの」を作ろう。

---

## どんなサービスを作る？

### アイデア例

自分の好きなことや、「こういうのあったらいいな」を考えてみて。

**何でもOK！自分が使いたいと思うものを作ろう。**

## GensparkかManusを選ぼう

| ツール | 特徴 | こんな人におすすめ |
|--------|------|------------------|
| **Genspark** | 検索するだけで作れる | 簡単に作りたい人 |

どっちでもOK！両方試してもOK！

# Genspark（ジェンスパーク）で作る方法

## 手順1: Gensparkを開く

1. ブラウザで「genspark.ai」を開く
2. ログインする（宿題0でやったはず）

## 手順2: 検索する

検索バーに、作りたいサービスに関連するキーワードを入力。

**例：**
- 「東京のラーメン店 比較 おすすめ」
- 「2024年の映画 ランキング レビュー」
- 「一人旅 おすすめスポット 国内」
- 「自宅でできる筋トレ メニュー 初心者」

## 手順3: Sparkpageを作る

1. 検索結果が出てきたら「**Create Sparkpage**」を押す
2. 数十秒待つ
3. サービスが完成！

## 手順4: カスタマイズする

チャット機能で変更をお願いできる：

- 「写真をもっと大きくして」
- 「価格帯で絞り込めるようにして」
- 「地図も追加して」
- 「もっとおしゃれなデザインにして」

---

# Manus（マヌス）で作る方法

## 手順1: Manusを開く

1. ブラウザで「manus.im」を開く
2. ログインする（宿題0でやったはず）

## 手順2: お願いを書く

チャット画面に、作りたいサービスを詳しく書く。

**テンプレート：**
```
〇〇のための△△サービスを作って。

機能:
- （できること1）
- （できること2）
- （できること3）

デザイン:（どんな見た目がいいか）
```

**例1: ラーメン店比較サービス**
```
ラーメン好きのためのラーメン店比較サービスを作って。

機能:
- エリア別にラーメン店を検索できる
- 「こってり」「あっさり」「辛い」などのタグで絞り込み
- お店の写真、メニュー、価格帯、営業時間が見れる
- お気に入り登録できる

デザイン: 熱くて美味しそうな感じ、赤と黒を基調に
```

**例2: 映画レビューサービス**
```
映画好きのための映画レビューサービスを作って。

機能:
- ジャンル別に映画を検索できる
- 公開年で絞り込める
- 映画のあらすじ、評価、レビューが見れる
- 見たい映画リストを作れる

デザイン: シネマっぽい雰囲気、ダークモードでおしゃれに
```

**例3: 旅行スポットサービス**
```
一人旅好きのための旅行スポットサービスを作って。

機能:
- エリア別にスポットを検索できる
- 「自然」「歴史」「グルメ」などのタグで絞り込み
- スポットの写真、アクセス、おすすめ度が見れる
- 旅行プランを保存できる

デザイン: 爽やかで冒険心をくすぐる感じ
```

## 手順3: 待つ

1. 送信ボタンを押す
2. Manusが自動で作り始める（見てるだけでOK）
3. 数分かかるから待っててね
4. 完成したらプレビューで確認！

## 手順4: 修正する

「ここ変えたい」と思ったら、追加でお願いすればOK：

- 「地図も表示して」
- 「口コミ投稿機能を追加して」
- 「スマホでも見やすくして」

# 提出するもの

## 提出物1: サービスのスクリーンショット

作ったサービスの画面を撮っておいてね。

**スクリーンショットの撮り方：**
- Windows: 「PrtSc」キーを押す
- Mac: 「Command + Shift + 3」を同時に押す

できれば2〜3枚（トップページ、検索結果、詳細ページなど）

## 提出物2: サービスのURL

作ったサービスのURLをメモしておいてね。

## 提出物3: 何のサービスを作ったか

例：「ラーメン店比較サービスを作りました」

## 提出物4: 見せた人の反応

例：
- 「すごいって言われた」
- 「これ使いたいって言われた」
- 「どうやって作ったの？って聞かれた」

---

# 提出のしかた

## 方法1: フォルダにまとめる

1. パソコンのデスクトップに「宿題1」フォルダを作る
2. スクリーンショットをそのフォルダに入れる
3. メモ帳で「メモ.txt」を作って、URL・サービス名・感想を書く
4. そのフォルダを先生に送る

## 方法2: LINEで送る

1. スクリーンショットを送る
2. URLを送る
3. 「〇〇サービスを作りました。△△に見せたら□□って言われました」と送る
