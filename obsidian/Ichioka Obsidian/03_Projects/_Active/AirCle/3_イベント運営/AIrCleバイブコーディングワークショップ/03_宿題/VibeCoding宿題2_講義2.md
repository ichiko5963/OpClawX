## 課題1: v0で10種類のUIデザインを作る（所要時間：約60分）

### この課題の目的

v0を使って、様々な種類のUIデザインを生成する練習をします。この課題を通じて「デザインを言葉で説明するスキル」を身につけてもらいます。バイブコーディングでは、自分のイメージをいかに正確にAIに伝えるかが重要です。

### v0とは何か

v0（ブイゼロ）は、Vercel社が提供するAIデザイン生成サービスです。自然言語（普通の日本語や英語）でデザインを説明すると、AIがReactコンポーネントとTailwind CSSを使ったコードを自動生成してくれます。

**Reactとは：**
Meta（旧Facebook）が開発したJavaScriptライブラリです。Webアプリケーションの画面（UI）を効率的に構築するために使われています。現在、最も人気のあるフロントエンド技術の一つです。

**コンポーネントとは：**
画面を構成する部品のことです。例えば「ボタン」「入力フォーム」「ナビゲーションバー」などが、それぞれ1つのコンポーネントになります。部品を組み合わせて画面全体を構築します。

**Tailwind CSS（テイルウィンドCSS）とは：**
CSSフレームワークの一種です。あらかじめ用意されたクラス名を使うだけで、デザインを適用できます。例えば `bg-blue-500` と書くだけで、背景色が青になります。

### 手順

**Step 1: v0にアクセスする**

1. ブラウザで `v0.dev` にアクセス
2. 右上の「Sign in」をクリック
3. Vercelアカウント（またはGitHubアカウント）でログイン

## 課題2: Google Antigravityでプロジェクトを作る（所要時間：約30分）

### この課題の目的

本格的な開発環境であるAntigravityに慣れることが目的です。AIエージェントと対話しながら、実際にコードが変更される様子を体験してください。

### Google Antigravityとは何か

Antigravityは、Googleが提供するクラウドベースの統合開発環境（IDE）です。VS Code（Visual Studio Code）をベースにしており、Gemini AIが統合されています。

**統合開発環境（IDE）とは：**
プログラムを書くためのソフトウェアです。コードエディタ（文章を書くためのメモ帳のようなもの）に加えて、ファイル管理、実行、デバッグ（バグの修正）などの機能が統合されています。

**クラウドベースとは：**
自分のパソコンにソフトウェアをインストールするのではなく、インターネット上（クラウド）で動作することを意味します。ブラウザさえあれば、どのパソコンからでも同じ環境で開発できます。

### 手順

**Step 1: Antigravityにアクセスする**

1. ブラウザで `idx.google.com`（Google IDX）にアクセス
   - AntigravityはGoogle IDXとして提供されています
2. Googleアカウントでログイン
3. ダッシュボード画面が表示されます

**Step 2: 新しいプロジェクトを作成する**

1. 「New Workspace」または「Create」をクリック
2. テンプレートから「React」を選択
3. プロジェクト名を入力（例：「my-first-app」）
4. 「Create」をクリック
5. 環境が構築されるまで待ちます（1〜2分）

**React（リアクト）とは：**
先ほども説明しましたが、Webアプリケーションのフロントエンド（ユーザーに見える部分）を構築するためのJavaScriptライブラリです。コンポーネントという部品を組み合わせて画面を作ります。

**Step 3: AIエージェントと対話する**

画面右側にAIエージェント（Gemini）のチャットパネルがあります。以下の3つの実験を順番に行ってください。

---

**実験1: シンプルな表示**

AIエージェントに以下のメッセージを送ります：

```
「Hello, Vibe Coding!」と表示するシンプルなページを作ってください。
文字は画面の中央に、大きく表示してください。
```

AIがコードを生成したら、右上の「Preview」ボタン（または再生マークのアイコン）をクリックして、結果を確認してください。

---

**実験2: インタラクション（操作）を追加**

続けて、以下のメッセージを送ります：

```
ボタンを追加してください。
ボタンをクリックしたら「こんにちは！バイブコーディング！」というアラートが表示されるようにしてください。
```

**アラートとは：**
ブラウザに表示される小さなポップアップウィンドウのことです。ユーザーに何かを通知したり、確認を求めたりするときに使います。

プレビューで動作を確認してください。ボタンをクリックして、アラートが表示されればOKです。

---

**実験3: スタイル（見た目）を変更**

最後に、以下のメッセージを送ります：

```
見た目を変更してください。
- 背景色を薄い青にする
- 文字の色を紺色にする
- ボタンをもっとおしゃれにする（角丸、影付き）
```

プレビューで変更が反映されていることを確認してください。

## 課題3: ターミナルで基本操作を覚える（所要時間：約20分）

### この課題の目的

「黒い画面」への苦手意識をなくすことが目的です。ターミナルは開発者にとって必須のツールですが、最初は怖く感じるかもしれません。基本的なコマンドを実行して、慣れていきましょう。

### ターミナルとは何か

ターミナル（または「コマンドライン」「CLI」とも呼ばれます）は、キーボードからコマンド（命令文）を入力してコンピュータを操作するツールです。

通常、私たちはマウスを使ってアイコンをクリックしてファイルを開いたり、フォルダを移動したりします。ターミナルでは、それと同じ操作を文字の命令で行います。

**なぜターミナルを使うのか：**
- 複雑な操作を自動化できる
- 開発ツールの多くがターミナルから実行される
- サーバー操作など、GUIがない環境でも操作できる
- 慣れると、マウス操作より速く作業できる

### 手順

**Step 1: ターミナルを開く**

Antigravityで、画面下部の「Terminal」タブをクリックします。黒い（または暗い色の）画面が表示されます。

もしタブが見つからない場合は、メニューから「View」→「Terminal」を選択するか、ショートカット `Ctrl + `` `（バッククォート）を押してください。

**Step 2: 基本コマンドを実行する**

以下のコマンドを、1つずつ入力して実行してください。各コマンドを入力したら、Enterキーを押して実行します。

---

**コマンド1: `ls`**

```
ls
```

**`ls` とは：**
「list」の略で、現在いるフォルダ（ディレクトリ）の中身を一覧表示するコマンドです。どんなファイルやフォルダがあるか確認できます。

実行すると、プロジェクト内のファイルやフォルダの名前が表示されるはずです。

---

**コマンド2: `pwd`**

```
pwd
```

**`pwd` とは：**
「print working directory」の略で、現在いるフォルダの場所（パス）を表示するコマンドです。「自分が今どこにいるか」を確認できます。

実行すると、`/home/user/my-first-app` のようなパスが表示されます。

---

**コマンド3: `cd` でフォルダを移動する**

```
cd src
```

**`cd` とは：**
「change directory」の略で、別のフォルダに移動するコマンドです。`cd フォルダ名` と入力すると、そのフォルダの中に移動します。

実行したら、`ls` を再度実行して、移動先のファイル一覧を確認してください。

---

**コマンド4: `cd ..` で親フォルダに戻る**

```
cd ..
```

`..` は「1つ上のフォルダ（親フォルダ）」を意味します。これで元の場所に戻れます。

---

**コマンド5: `cat` でファイルの中身を見る**

```
cat package.json
```

**`cat` とは：**
ファイルの内容をターミナル上に表示するコマンドです。`cat ファイル名` と入力します。

**package.jsonとは：**
Node.js（JavaScriptの実行環境）のプロジェクトで使われる設定ファイルです。プロジェクトの名前、バージョン、使用しているライブラリなどの情報が記載されています。

---

**Step 3: Claude CLIを試す（オプション）**

AntigravityにClaude（Anthropic社のAI）がインストールされている場合、以下のコマンドで呼び出せます。

```
claude
```

Claudeが起動したら、挨拶してみてください：

```
こんにちは！私はプログラミング初心者です。ターミナルの使い方を勉強しています。
```

Claudeとの会話を終了するには、`exit` と入力するか、`Ctrl + C` を押してください。

**注意：** Claude CLIが利用できない場合もあります。その場合は、GeminiのAIエージェント（画面右側）で同様の会話ができます。

- Googleアカウントが有効か確認してください
