### スライド 3: 天気APIと何が違う？

**タイトル**: 事実 vs 思考 **内容**:

- **天気API**: 「今の気温は？」→「25度です」（事実を返すだけ）
    
- **Gemini API**: 「やる気出ないんだけど...」→「じゃあ、5分だけやってみませんか？」（**考えて、生成する**）
    
- **違い**: 決まった答えがない。文脈を理解する。 **発表者ノート**: 天気APIは「新聞配達」ですが、Gemini APIは「コンサルタント」です。相談に乗ってくれます。

**タイトル**: 脳みその調達場所 **内容**:

### スライド 5: 【演習】API Keyの取得

**タイトル**: 🛠️ 脳みその契約書 **手順**:

1. `aistudio.google.com` にアクセス（Googleアカウントでログイン）。
    
2. **「Get API key」** をクリック。
    
3. **「Create API key」** でキーを発行し、コピーする。 **発表者ノート**: これが「知能へのアクセス権」です。絶対に他人に教えてはいけません。
    

### スライド 6: 環境変数への登録

**タイトル**: 🛠️ 鍵の保管（復習） **内容**:

- Lecture 10と同じ手順。
    
- Antigravityの `.env` ファイルを開く。
    
- `VITE_GEMINI_API_KEY=さっきのキー` を追加。
    
- **注意**: `VITE_` をつけないと、Reactから読み込めないことが多い（※Viteの場合）。 **発表者ノート**: ここをミスると動きません。おまじないだと思って `VITE_` をつけてください。

## 【Part 5: 発展（JSONモード）】

### スライド 14: 文章だけじゃ物足りない

**タイトル**: 言葉より「行動」を **内容**:

- 励ましも嬉しいけど、**「タスクを分解」** してくれたらもっと便利。
    
- 例：「カレーを作る」→ 「肉を買う」「野菜を切る」「煮込む」に分解して、**自動でリストに追加**してほしい。
    
- そのためには、AIに「日本語」ではなく **「データ（JSON）」** で返してもらう必要がある。 **発表者ノート**: ここからが中級テクニックです。AIを「チャットボット」ではなく「データ生成エンジン」として使います。
    

### スライド 15: 構造化出力（JSON Mode）

**タイトル**: AIに枠をはめる **内容**:

- プロンプトの最後にこう付け加える： **「回答は必ずJSON形式にしてください。例：`[{"task": "野菜を切る"}, ...]`」**
    
- こうすると、プログラムがAIの答えを読み取って、そのままアプリの機能として使える。 **発表者ノート**: AIの答えを「読む」だけでなく「機能として使う」ための魔法です。
    

### スライド 16: 【演習】タスク分解機能

**タイトル**: 🛠️ 「細かくして」ボタン **指示**: **「各タスクの横に『分解』ボタンをつけて。 押すと、そのタスクを3つのサブタスクに分解して、 自動的にTodoリストに追加（Supabaseに保存）するようにして。」** **発表者ノート**: これができれば、もう初心者卒業です。「AIが考えた手順」が、そのまま「アプリのデータ」になります。

## 【Part 6: チューニングとコスト】

### スライド 17: トークンとは？

**タイトル**: AIの燃料 **内容**:

- AIは「文字数」でお金がかかる（今回は無料枠だけど）。
    
- **入力（Prompt）** と **出力（Response）** の合計で計算される。
    
- **教訓**: 無駄に長いプロンプトは送らない。必要な情報だけ送る。 **発表者ノート**: タスクリストを送る時、「完了済みのタスク」まで送っていませんか？ AIにアドバイスをもらうなら「未完了」だけで十分かもしれません。節約の意識を持ちましょう。
    

### スライド 18: ストリーミング（Streaming）

**タイトル**: 待たせない技術 **内容**:

- 今の実装だと、AIが書き終わるまで3秒くらい待たされる。
    
- **ストリーミング**：AIが喋り出した瞬間に、文字をパラパラと表示する技術（ChatGPTみたいな動き）。
    
- Antigravityのエージェントに **「回答をストリーミング表示にして」** と言えば実装できる。 **発表者ノート**: これを入れるだけで、アプリの「サクサク感」が段違いになります。余裕があればやってみましょう。
