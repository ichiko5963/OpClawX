## 【Part 2: Vercel連携】

### スライド 4: Vercelとは？

**タイトル**: Next.jsの生みの親 **内容**:

- Next.jsを開発している会社が作った、Web公開サービス。
    
- **特徴**:
    
    - **設定ゼロ**：Next.jsなら自動で認識して最適化してくれる。
        
    - **爆速**：世界中にサーバーがある。
        
    - **無料**：個人開発なら十分すぎる無料枠。 **発表者ノート**: Next.jsで作ったなら、Vercel一択です。iPhoneとMacくらい相性が抜群です。
        

### スライド 5: 【演習】Vercelアカウント作成

**タイトル**: 🛠️ 店を構える **手順**:

1. `vercel.com` にアクセス -> **「Sign Up」**。
    
2. **「Continue with GitHub」** を選ぶ（これが重要！）。
    
3. ログインすると、ダッシュボードが表示される。 **発表者ノート**: GitHubアカウントでログインすることで、「倉庫」と「店舗」のコネクションが完了します。
    

### スライド 6: 【演習】プロジェクトのインポート

**タイトル**: 🛠️ 商品の陳列 **手順**:

1. **「Add New...」** -> **「Project」**。
    
2. **「Import Git Repository」** のリストに、前回作った `my-vibe-app` があるはず。
    
3. **「Import」** ボタンを押す。 **発表者ノート**: 倉庫にある荷物を選んで「これを店に出すよ」と指示します。まだ「Deploy」は押しちゃダメですよ！

## 【Part 3: 最後の難関「環境変数」】

### スライド 7: そのままDeployしてはいけない

**タイトル**: ⚠️ 鍵を忘れていませんか？ **内容**:

- ここで「Deploy」を押すと、**エラーになります（または動きません）。**
    
- なぜ？ -> **「鍵（.env）」がGitHubにないから。**
    
- セキュリティのため、鍵は倉庫（GitHub）には置かないルールでした。
    
- **店舗（Vercel）の金庫に、直接鍵を入れる必要があります。** **発表者ノート**: 初心者が一番ハマる罠です。「ローカルでは動いたのに！」の原因は99%これです。
    

### スライド 8: 環境変数の入力

**タイトル**: 🛠️ 金庫に鍵を入れる **ビジュアル**: **手順**:

1. Import画面にある **「Environment Variables」** をクリックして開く。
    
2. Antigravityの `.env` ファイルの中身をコピペする。
    
    - Name: `VITE_SUPABASE_URL`, Value: `https://...`
        
    - Name: `VITE_GEMINI_API_KEY`, Value: `AIza...`
        
3. 1つずつ「Add」を押して登録する。 **発表者ノート**: SupabaseやGeminiの鍵を、一つずつ丁寧に登録してください。これがないと、本番環境でデータベースに接続できません。
    

### スライド 9: デプロイ実行！

**タイトル**: 🛠️ スイッチオン **手順**:

1. 鍵の設定が終わったら...
    
2. **「Deploy」** ボタンをクリック！
    
3. 画面が切り替わり、ビルド（建築）が始まる。
    
4. 1〜2分待つ...。 **発表者ノート**: さあ、祈りましょう。裏側では、Vercelのロボットたちが猛スピードであなたのアプリを組み立てています。

## 【Part 5: 運用と自動化（CI/CD）】

### スライド 12: 修正したくなったら？

**タイトル**: 魔法の更新術 **内容**:

### スライド 13: CI/CD（継続的デリバリー）

**タイトル**: 自動化された店舗 **内容**:

- GitHubにPushされた瞬間、Vercelがそれを検知する。
    
- 「お、新しい荷物が届いたな。店に並べ直そう」
    
- **勝手に再デプロイが走り、数分後には本番サイトが更新される。**
    
- これが現代の開発フロー **「CI/CD」** です。 **発表者ノート**: あなたが寝ていても、Pushさえすれば世界中のサーバーが更新されます。
