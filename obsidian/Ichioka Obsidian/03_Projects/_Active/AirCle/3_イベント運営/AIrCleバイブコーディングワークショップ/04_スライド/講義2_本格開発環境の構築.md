# 講義2: 本格開発環境の構築

## デザインから開発まで、プロ級のツールを手に入れる

---

# 今日できるようになること

1. 🎨 **v0** でプロ級のデザインを作る
2. 🏗️ **Antigravity** をインストールして開発環境を整える
3. 💻 **ターミナル** の恐怖を克服し、AIツールを使い分ける

---

# 人は見た目が9割

- 機能がすごくても、見た目がショボいと使われない
- でも、デザインの勉強にはセンスと時間がいる...
- **解決策：AIに「カッコよくして」と頼む**

---

# Vercel社が生んだ「Generative UI」

- 画像生成AI（Midjourneyなど）の「Webサイト版」
- **「こんな画面作って」と言うと、絵ではなく「コード」で画面を作ってくれる**
- 作られるのは、現代最強の技術セット（React + Tailwind CSS）

---

# 🛠️ v0の世界へ

1. `v0.dev` にアクセス
2. **「Log in」** をクリック（Vercelアカウントが必要）
3. 無料プランで始められる

---

# 🛠️ 言葉を形にする

**プロンプト例**:
```
iPhoneの計算機アプリのような、モダンでミニマルな電卓を作って。
ボタンは丸く、配色は黒とオレンジで。
```

---

# A案、B案、C案

- v0は一度に3種類のデザイン案を出してくれる
- あなたの仕事は **「選ぶ（Curating）」** こと
- 「B案の雰囲気が好きだな」→ Bをクリック

---

# 🛠️ もっとワガママに

- 選んだ案に対して、追加で指示を出す
- **「背景をダークモードにして」**
- **「数字をもっと大きく、太くして」**
- **「ボタンに影（シャドウ）をつけて立体的にして」**

---

# 正体は「メモ帳」

- 文章を書くソフト ➡ **Word**
- 表計算をするソフト ➡ **Excel**
- プログラム（コード）を書くソフト ➡ **VS Code**
- **これだけの違いです。**

---

# VS Code + AI

- **VS Code**: 便利なメモ帳。
- **Antigravity**: そのメモ帳に、**GoogleのAI（Gemini）が最初から住んでいる**。
- 自分で設定しなくても、最初からAIが手伝ってくれる特別版です。

---

# 🛠️ 入手する

1. `antigravity.google` にアクセス
2. **「Download for Desktop」** をクリック
3. インストーラーを起動

---

# コックピットの構造

1. **左側（サイドバー）**: **「フォルダの中身」**。PCのファイル一覧と同じ。
2. **真ん中（エディタ）**: **「書く場所」**。ここに文字を打つ。
3. **下側（ターミナル）**: **「命令する場所」**。AIやPCへのチャット欄。

---

# 執事を呼び出す

- 画面右側のパネルを開く。
- ここが **「AIへの指示出しチャット」** です。
- v0で作ったデザインのコードをコピーして、ここに貼り付けて「これで作って」と頼む

---

# パソコンとの「LINE」です

- **GUI（マウス操作）**: レストランでメニューを指差して注文する。「これください」
- **CLI（ターミナル）**: 厨房のシェフに直接LINEする。「メニューにないけど、豚肉を焼いて塩コショウして！」
- **ターミナル = パソコンへの「直通チャット」**

---

# マウスより「言葉」が強い

- マウスだと「ファイルを作って、名前を変えて、移動して...」と3回操作が必要。
- ターミナルなら **「これやっといて」の一言（1行）** で終わる。
- **特にAIとの相性が抜群！**（AIは言葉が得意だから）

---

# どれを使えばいいの？

- **Claude Code** = 現場監督（全体構築）
- **Codex** = 外科医（修正・エラー）
- **Gemini CLI** = 巨人（大量データ）

---

# 🛠️ 現場監督を召喚

1. Antigravityのターミナルを開く
2. `claude` と入力してEnter
3. 「何を作りますか？」と聞かれるので、日本語で答える

---

# 🛠️ チャット形式で進行

- 黒い画面の中で、ClaudeとLINEする感覚
- あなた：「Reactで簡単な電卓アプリを作って」
- Claude：「わかりました。ファイルを作成します...完了しました」
- **マウスを使わずにアプリができた！**
