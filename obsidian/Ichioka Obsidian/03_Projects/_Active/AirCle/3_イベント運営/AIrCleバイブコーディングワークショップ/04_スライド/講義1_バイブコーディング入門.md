# 講義1: バイブコーディング入門

## AIに「作って」と頼むだけ。今日、カフェ比較サービスを作ります。

---

# 正直、プログラミングって怖くない？

- 黒い画面に英語がズラーッ...
- エラーが出たらパニック...
- 何年も勉強しないと無理...？

---

# AIのおかげで、全部変わった

- 昔 → ホームページ作るだけでも何ヶ月も勉強
- 今 → **「サービス」が数分で作れる**

---

# 今日できるようになること

- ホームページじゃない、**サービス**を作る
- 「カフェ比較サービス」を実際に作る
- スマホで動く、使えるものを作る

---

# ホームページ vs サービス

| | ホームページ | サービス |
|---|------------|---------|
| 例 | 会社紹介、お店の案内 | 食べログ、メルカリ、Instagram |
| 特徴 | 見るだけ | **使える**、**動く** |
| できること | 情報を読む | 検索、投稿、比較、予約... |

---

# カフェ好きのためのカフェ比較サービス

- カフェの情報を比較できる
- おすすめのカフェを紹介
- エリアやジャンルで絞り込み
- **カフェ好きが本当に使いたいサービス**

---

# 今日使う魔法のツール

1. **Genspark** - 検索するだけでサービスが作れる
2. **Manus** - 言葉だけでアプリを全自動生成
3. **v0** - プロ級のUIデザインを作る
4. **Gemini** - AIと相談しながら作る

---

# Genspark = 検索するだけでサービスを作る

- 普通の検索 → 検索結果のリストが出る
- Genspark → **まとまったサービス**が出てくる
- 検索 + ボタン1つで完成

---

# 【実践】Gensparkでカフェサービスを作る

1. 「genspark.ai」を開く（宿題0でログイン済みのはず）
2. 検索バーに入力...

---

# 【実践】検索するだけ

**検索例**:
```
東京のおしゃれなカフェ 比較 おすすめ
```

**手順**:
1. 検索結果が出てきたら「Create Sparkpage」を押す
2. 数十秒待つ
3. カフェ比較サービスが完成！

---

# 【実践】自分好みに変える

- 「京都のカフェも追加して」
- 「価格帯で絞り込めるようにして」
- 「写真をもっと大きくして」
- 「インスタ映えするカフェだけ表示して」

---

# Manus = 言葉だけでサービスを全自動生成

- 「こんなサービス作って」と言うだけ
- AIが勝手にファイルを作り、コードを書く
- **見てるだけで完成**

---

# 【実践】Manusでカフェサービスを作る

1. 「manus.im」を開く（宿題0でログイン済みのはず）
2. チャット画面に入力...

---

# 【実践】詳しくお願いする

```
カフェ好きのためのカフェ比較サービスを作って。

機能:
- エリア別にカフェを検索できる
- 「静か」「作業向き」「インスタ映え」などのタグで絞り込み
- カフェの写真、メニュー、価格帯、営業時間が見れる
- お気に入り登録できる

デザイン: おしゃれで、カフェ好きが使いたくなる感じ
```

---

# 【実践】Manusが自動で作る

- Manusが勝手にファイルを作り始める
- コードを書いている（見てるだけでOK）
- 数分で完成
- **プレビューで実際に動くサービスが見れる**

---

# v0 = プロ級のデザインを作る

- 「v0.dev」でUIデザインを作れる
- 「カフェ比較サービスのUI」と入力するだけ
- プロのデザイナーが作ったようなデザインが出てくる
- コードも一緒に出してくれる

---

# Gemini = AIと相談しながら作る

- 「gemini.google.com」→「キャンバス」
- 「カフェ比較サービスのアイデア出して」
- 「このサービスに何の機能があったらいい？」
- アイデア出しや相談に使える

---

# 【実践】作ったサービスをスマホで見る

1. 作ったサービスのURLをコピー
2. LINEかメールで自分のスマホに送る
3. スマホで開いてみる
4. 実際に使ってみる

---

# 今日できるようになったこと

1. **ホームページじゃなくて「サービス」が作れた**

# 宿題と次回予告

**宿題**:
- GensparkかManusで「自分だけのサービス」を1つ作る
- テーマは自由（カフェ以外でもOK）
- 家族か友達に見せる
- スクリーンショットとURLを提出

**次回予告**:
- 次回は「本格的な開発環境」を整えます
- もっと自由に、もっと本格的なアプリを作れるようになる
