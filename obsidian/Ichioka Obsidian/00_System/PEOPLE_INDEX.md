## 表記ゆれルール
- **正式名**: フルネームまたは最も使われる表記
- **別名**: Slack表示名、ニックネーム、漢字/英字バリエーション
- 検索時は別名も含めてマッチする

## 人物フォルダ構造
```
10_People/
└── [人物名]/
    ├── PROFILE.md    # 確定情報のみ
    └── NOTES.md      # 推測・メモ（未確定）
```

## 注意
- PROFILE.md には**確定した事実のみ**を記載
- 推測（役職推定、関係性推定など）は NOTES.md へ
- 議事録は人物フォルダに置かない → プロジェクトの MTG/ へ
