### email_manager.py タイムアウト機能追加 (12:54-13:00)
- **問題**: email_manager.py が30秒以上ハングする
- **解決策**:
  1. coreutils インストール（`brew install coreutils`）→ `gtimeout` コマンド利用可能に
  2. スクリプトに30秒タイムアウト組み込み（`signal.alarm(30)`）
  3. ハング時は自動終了（exit code 124）
- **結果**: テスト成功、30秒以内に正常完了
