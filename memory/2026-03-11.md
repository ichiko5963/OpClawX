# 2026-03-11 Memory

## Viral Post Automation v2.0 (OpClawX) Released

Built a complete viral post automation system for X/Twitter and pushed to GitHub.

### Repository
- **URL:** https://github.com/ichiko5963/OpClawX
- **Name:** OpClawX
- **Status:** Public, MIT License

### System Components
- `core/analyzer.js` — 15 viral pattern detection engine (CSV/JSON parsing)
- `core/generator.js` — Prompt generation for LLMs (ChatGPT/Claude/Gemini)
- `scheduler/daily.js` — Daily delivery via Discord/Slack webhook
- `i18n/index.js` — 5-language support (EN/JP/CN/KR/ES)
- `cli.js` — Unified CLI: analyze | generate | schedule | suggest
- `web/index.html` — Zero-dependency web dashboard
- `.github/workflows/daily.yml` — GitHub Actions for automation

### 15 Viral Patterns Implemented
1. Breaking News (速報型)
2. Save for Later (保存版型)
3. Global Trend (海外バズ型)
4. Conclusion First (結論型)
5. Honest Opinion (正直型)
6. VS Battle (比較型)
7. First Impression (体験記型)
8. By The Numbers (数字強調型)
9. Insight (洞察型)
10. Free Resource (配布型)
11. Pro Tips (裏技型)
12. Warning (警告型)
13. Storytelling (ストーリー型)
14. Complete Guide (完全解説型)
15. Future Forecast (予測型)

### Key Features
- Numbers/CSV file ingestion
- Pattern detection with engagement scoring
- Auto-language detection (Japanese/Chinese/Korean/English/Spanish)
- Daily morning URL delivery to Discord/Slack
- Multilingual prompt generation
- 16 passing tests

### Usage Flow
```
node cli.js analyze -f posts.csv --lang ja
node cli.js schedule --add --time 07:00 --lang ja --topic "AIニュース" --webhook <url>
```
