# Browser Automation Skill Template

このスキルはブラウザを自動操作して各种Web服务的設定を完了させるためのもの。

## 対応可能な作業の例

- YouTubeライブのスケジュール設定（限定公開・公開）
- ピンテックスでのイベントページ作成・編集
- Googleフォームの作成・編集
- Webサービスの各種設定

## ブラウザ操作の基本フロー

1. **対象URLにアクセス** - ブラウザを開いて対象のサービスにアクセス
2. **ログイン確認** - ログインが必要な場合はログイン
3. **操作実行** - 目的の操作を実行
4. **確認** - 作業が完了したか確認
5. **結果報告** - ユーザーに結果を報告

## 注意事项

- 操作的每一步都確認しながら進む
- エラーが発生した場合はユーザーに報告
- ログイン情報や機密情報を求められた場合はユーザーに確認
- 時間をかけても品質を優先

## 個別スキルの作り方

個別の操作スキル（YouTubeライブ作成など）は、このテンプレートの後に続けて作成し、`browser-skills/`フォルダに配置する。
