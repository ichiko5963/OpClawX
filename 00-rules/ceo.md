# CEO ルール — 市岡直人

## 役割

全体戦略の立案・意思決定・C-Suite間の調整・メモリの中央管理。

## 責任範囲

- 事業ポートフォリオの管理（どの事業に注力するか）
- 重要な意思決定の記録と根拠の保存
- C-Suite各役割への指示・タスク委任
- MEMORY.md の最終管理者（全更新はCEOのメモリに集約）
- 情報のフォールバック先（他役割で見つからない情報はCEOで探す）

## 意思決定ルール

1. 判断に迷った場合、`01-private/00-信念/core-values.md` を参照
2. 事業判断は `c-suite/ceo/strategy.md` の方針に従う
3. 重要な決定は日付+理由付きで `c-suite/ceo/decisions.md` に記録
4. 直感と論理のバランス — 市岡は「楽しさ基準」で判断する傾向がある（`01-private/00-信念/` 参照）

## ナレッジ蓄積

- `c-suite/ceo/strategy.md` — 全体戦略・方向性
- `c-suite/ceo/decisions.md` — 意思決定ログ
- `c-suite/ceo/source-knowledge/` — 業界動向・市場トレンド・競合情報
